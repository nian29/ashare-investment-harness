# 宏观传导机制

> 配合 [[investment-handbook]] 第四部分（宏观环境扫描）+ [[financial-crisis-classics]] 使用。
> 宏观不是"大道理"，是直接作用在你每一分钱持仓上的力量。这篇讲清楚从利率到你的具体持仓，中间经过了哪些环节。

---

## 一、利率是万有引力

所有资产定价的底层公式都是 DCF——利率决定了折现率 r，r 决定了资产价格。

```
利率 ↑ → r ↑ → DCF 分母变大 → 资产价格 ↓
利率 ↓ → r ↓ → DCF 分母变小 → 资产价格 ↑
```

**但不同资产对利率的敏感度完全不同**。这就是为什么"加息杀科技股，但不一定杀银行股"。

### 各类资产对利率的敏感性

| 资产类型 | 利率敏感性 | 为什么 |
|---------|----------|--------|
| 成长股/科技 | 🔴 极高 | 大部分价值在遥远未来（终值占比 80%+），利率轻轻一动终值剧烈波动 |
| 消费龙头 | 🟡 中等 | 利润稳定、近期现金流占比大，对折现率变化不敏感 |
| 银行 | 🟡 复杂 | 有两条传导路径，方向相反（见下一节） |
| 电力/公用事业 | 🟡 中等 | 重资产+长期现金流，类似长久期债券 |
| 短期国债 | 🟢 极低 | 到期短，折现效应最小 |

**当前应用**（2026.6）：
```
美联储 6/16-17 议息 → 市场预期 12 月加息概率 60-68%
如果加息落地 → r ↑ → 全球成长股/科技股承压最大
→ 你的 CPO 相关标的会受到冲击
→ 但农行和海天受影响小（利润稳定、近期现金流占比大）
→ 电力 ETF（重资产+长期现金流）会受到中等冲击
```

## 二、利率→银行→股市的完整传导链

这是最重要的传导路径，因为银行是你最大的仓位。

### 链条拆解

```
美联储加息
  → 美国国债收益率 ↑
  → 全球资金回流美国（买更高收益的美债）
  → A 股外资流出（北向资金净流出）
  → A 股整体估值承压

同时：
  → 中国央行面临汇率贬值压力
  → 央行可能被动跟随收紧（或至少不放松）
  → 国内利率上行（LPR ↑）
  → 银行净息差（取决于存款利率和贷款利率谁涨得快）
```

### 加息对银行的双重效应

**效应 1（利好）**：贷款利率涨得比存款利率快 → 净息差扩大
**效应 2（利空）**：利率上行 → 经济放缓 → 坏账增加 → 信用成本上升

```
当前（2026.6）判断：
→ 农行 NIM 在 1.28% 历史低位
→ 如果利率上行 → NIM 有修复空间（短期利好）
→ 但如果经济放缓 → 零售不良率可能继续恶化（中长期利空）
→ 综合：利率小幅上行对银行未必是坏事，大幅上行才是
```

## 三、信用周期——最重要的宏观领先指标

信用周期比 GDP 周期领先 1-2 个季度——它是宏观的"先行指标"。

### 三个关键数据

| 数据 | 含义 | 去哪看 |
|------|------|--------|
| **社融增量** | 实体经济的"借钱量"。上升 = 经济要扩张 | 央行每月 10-15 号发布 |
| **M1-M2 剪刀差** | M1 增速 > M2 增速 = 企业活期存款增加 = 企业准备花钱投资 | 央行同步发布 |
| **LPR** | 银行贷款基准利率。下行 = 央行在放松 | 央行每月 20 号发布 |

### M1-M2 剪刀差的用法

```
M1（活期）> M2（定期+活期）增速 → "正剪刀差"
  → 企业和居民把钱从定期转到活期
  → 准备投资/消费
  → 经济转暖的前置信号
  → 利好：消费、银行（贷款需求上升）

M1 < M2 → "负剪刀差"
  → 企业和居民把钱锁在定期（对未来悲观）
  → 经济转冷的前置信号
  → 利空：消费、银行（贷款需求下降）
  → 利好：债券（避险情绪）
```

## 四、你的持仓对宏观变量的敏感性矩阵

| 宏观变量 | 农行 | 海天 | 电力ETF | 核电 |
|---------|------|------|---------|------|
| **利率 ↑** | 复杂（↑↓） | 轻微 ↓ | ↓（重资产折现） | ↓ |
| **利率 ↓** | 复杂 | 轻微 ↑ | ↑ | ↑ |
| **煤价 ↑** | — | — | ↓（火电成本） | — |
| **煤价 ↓** | — | — | ↑（火电利润） | — |
| **油价 ↑** | — | ↑（原料+运输） | ↓ | — |
| **油价 ↓** | — | ↓（利好） | ↑ | — |
| **经济复苏** | ↑（贷款增+坏账降） | ↑（餐饮回升+酱油消费增） | ↑（用电量增） | ↑ |
| **经济衰退** | ↓（坏账增+NIM降） | ↓（消费降级） | ↓（用电减） | 轻微 ↓（刚需） |
| **人民币贬值** | — | — | — | — |
| **财政刺激** | ↑（基建贷款） | ↑（消费刺激） | ↑ | ↑ |
| **电价改革** | — | — | ↑（利好核电） | ↑↑ |

**关键洞察**：你的四个持仓之间存在一定的对冲——

```
煤价跌 + 油价跌 = 利好电力 ETF（火电成本降）+ 利好海天（原料成本降）
但这个组合对"利率上行"整体偏敏感（银行复杂 + 电力受影响）
```

## 五、美联储决策框架——一份能用的速查

你的手册里有关键日期，但缺少"怎么看美联储决议"的框架。

### 议息会议四件套

| 内容 | 重要性 | 看什么 |
|------|--------|--------|
| **利率决议** | ⭐⭐⭐ | 加/降/不动 + 幅度 |
| **点阵图** | ⭐⭐⭐ | 委员们对未来利率的预测分布。**比决议本身更重要** |
| **声明措辞** | ⭐⭐ | "保持耐心"vs"即将行动" → 语气变化是领先信号 |
| **鲍威尔（沃什）发布会** | ⭐⭐⭐ | 30 分钟问答，常常比声明更有信息量 |

### 点阵图怎么看

```
点阵图 = 19 位 FOMC 委员各自对年末利率的预测
→ 中位数（第 10 位）= 市场的核心关注点
→ 如果中位数比上次上移 → 鹰派（更多委员预计要加息）
→ 如果中位数比上次下移 → 鸽派（更多委员预计要降息）
→ 分布越分散 → 内部分歧大 → 未来路径不确定性高
```

### 议息后市场反应的规律

```
决议公布后 5 分钟：波动最大（算法交易在抢跑）
30 分钟后：方向开始明确（人类交易员读完决议）
发布会期间：可能有二次波动（问答环节出意外信息）
收盘：当天方向基本定调

不要在前 5 分钟交易。让市场消化 30 分钟再判断。
```

## 六、宏观与微观的结合——使用这个框架的正确方式

**不要做宏观预测。** 马克斯和芒格反复提醒：没有人能持续预测宏观。

**要做的事**：
```
1. 知道当前的宏观环境（利率方向、信用周期位置、政策基调）
2. 知道你的持仓对哪些宏观变量最敏感（对照第五章矩阵）
3. 在极端时逆向——市场因宏观恐慌砸盘时，如果基本面没变，是你最好的买点
4. 不用宏观来做日常择时，用它来识别"市场是否在定价极端情景"
```

---

**Why:** 用户手册里有宏观关注点但没有传导机制——知道要关注美联储，但不知道为什么关注、关注决议里的哪个部分、利率变化怎么传导到自己的持仓。

**How to apply:** 每次议息后对照点阵图判断利率方向 → 用敏感性矩阵评估对四只持仓的影响 → 如果市场因宏观恐慌过度反应，是逆向买入的窗口。
