# 交易决策日志

> 每次买入/卖出前填写，决策后追加。不做笔记不交易。

---

## 使用方式

```
每次交易前复制下面的模板，填写后追加到本文件底部。
最新记录在最上面。
```

---

## 模板

```markdown
## [买入/卖出] STOCK_NAME (SYMBOL) — YYYY-MM-DD

### 操作
- 方向: 买入 / 卖出
- 数量: ___ 股
- 价格: ___ 元
- 金额: ___ 元
- 操作后仓位: ___%

### 决策前检查（必须逐条回答）
1. 买入逻辑是什么？（一句话）
2. 事前验尸：假设 2 年后亏了 50%，最可能因为什么？（写 3 个原因）
   - 
   - 
   - 
3. 反向 DCF：当前价格在赌什么？
4. 我比市场多知道什么？
5. 博弈论检查：谁在另一边？他们为什么卖/买？
6. 操作后仓位是否超过 20%？（是 → 必须有减仓计划）
7. 今天涨跌幅是否 > 5%？（是 → 不追）
8. 是执行纪律还是情绪反应？

### 决策前心理状态
- 当前情绪: 冷静 / 焦虑 / 兴奋 / 恐慌
- 是否有 FOMO（害怕错过）: 是 / 否
- 是否受他人影响: 是 / 否

### 操作后记录
- 实际成交价: ___ 元
- 滑点: ___ 元
- 操作后感受: ___

### 复盘（操作 1 周后填写）
- 回头看，这个决策对吗？
- 学到了什么？
```
