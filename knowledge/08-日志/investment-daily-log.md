## 📅 2026-06-22（周一）端午后首日 · 科技继续狂欢，传统加速失血

### 🔍 市场温度判断（框架：[[financial-crisis-classics]]）

**当前处于 Kindleberger 狂热阶段，且狂热的"聚焦效应"在加剧。** 信号：

- 节后首日延续"深强沪弱"：上证微跌 0.21%，但科创 50 节前已暴涨 3.84% 创历史新高
- 保险板块暴跌 5%+，中国太保、新华保险、中国平安集体下挫——资金加速从传统板块撤离
- 银行、白酒等传统权重遭集中抛售，而培育钻石（黄河旋风一字涨停）、锆概念（长裕集团 5 天 4 板）、半导体设备（晶升股份涨超 10%）继续疯狂
- 早盘半小时成交额超 6000 亿，较上日放量 864 亿——**资金不是在撤退，是在换仓**
- 叠加三个短期压力：股指期货交割日 + 半年末 MPA 考核 + 549 亿解禁潮

**与 6/18 的对比**：6/18 我判断为"狂热中后期，出现裂缝"。节后第一天看，裂缝没有扩大成恐慌，而是变成了**更极端的结构分化**——科技从"强势"变成了"吸金黑洞"。资金不是离场，是从传统板块被抽到科技。这对我们的防御型组合（银行+消费+电力+高速）构成了系统性逆风。

**霍华德·马克斯的框架**：这个阶段的特点是"好的东西越来越好，差的东西越来越差"——不是因为基本面，而是因为资金流向自我强化。我们的持仓不是"差"，而是不在当前市场的审美方向上。

### 🏛️ 宏观传导分析——沃什鹰派被科技狂热完全无视（框架：[[macro-transmission-mechanism]]）

6/18 沃什鹰派首秀（废除前瞻指引 + 9 票支持加息 + 点阵图中位数 3.8%）按理应该压制科技估值。但实际上：

```
沃什鹰派 → 利率↑ → DCF折现率↑ → 科技估值应承压
    ↓
    实际：科技继续暴涨，传统板块反被抽血
    ↓
    原因：国内政策对冲力 > 海外利率压力
      ├── 八部门"人工智能+消费"意见
      ├── 七部门算力资源开放
      ├── 大基金持续输血半导体
      └── 陆家嘴论坛政策预期仍在发酵
```

**对持仓的传导（更新版）**：

| 持仓 | 宏观逻辑 | 风格逻辑（更强） | 实际表现 |
|------|---------|----------------|---------|
| 农行 | 加息→NIM改善 🟢 | 资金从银行→科技 🔴 | -0.31%，风格逻辑压倒宏观 |
| 海天 | 利率影响中性 🟡 | 消费不在风口 🔴 | -1.97%，离止损仅1.64% |
| 电力ETF | 利率↑→长久期资产承压 🔴 | 电力不在风口 🔴 | 6/18 -3.67%，双重利空 |
| 宁沪 | 利率↑温和偏空 🟡 | 防御被抛弃 🔴 | -1.24%，相对抗跌 |

**核心结论**：当前市场的定价权在"国内产业政策 + 风格动量"，不在"宏观利率逻辑"。我们的组合押注的是后者（利率→银行受益、防御→资金回流），但市场走的是前者。需要接受这个短期现实——**基本面逻辑正确不等于短期股价正确**。

### 🏛️ 宏观传导分析——沃什鹰派首秀的冲击（框架：[[macro-transmission-mechanism]]）

**今日核心宏观事件**：北京时间 6/18 凌晨，美联储新任主席沃什首次议息——**比市场最鹰的预期还鹰**。

**决议核心要点**：

| 项目 | 细节 |
|------|------|

| 通胀预期 | PCE 2.7→3.6%, 核心 PCE 2.7→3.3% |
| 市场反应 | 美股跳水（道指从高点跌 500+点）、黄金日内波动 $150+、美元指数拉升百点、2Y 美债 +15bp |

**沃什变革的深层含义**：
> 这不是一次普通的鹰派决议。沃什在**改变美联储的整个沟通框架**——废除前瞻指引意味着未来每次决议都是"开盲盒"，市场无法提前定价。这增加了所有资产的波动率溢价（volatility premium），对 A 股意味着：外资流入更谨慎、汇率波动更大、科技股估值折现率更高。

**宏观传导链对持仓的影响**（框架第 3 章）：

```
沃什鹰派 → 加息预期升温
  ├── 美元走强 → 人民币贬值压力
  │   ├── 外资流出压力 → A 股整体承压
  │   └── 但银行股例外（见下）
  ├── 利率↑ → NIM 改善预期 → 🟢 银行利好
  │   └── 传导：加息→贷款收益率↑→NIM扩张→银行利润↑
  │   └── 这就是今天农行主力净流入 3.93 亿的宏观逻辑！
  ├── 利率↑ → DCF 折现率↑ → 🔴 科技/成长承压
  │   └── 但 A 股科技有陆家嘴论坛政策对冲（吴清 20 条+大基金）
  │   └── 今天科技没崩，但"鹰派+长假"组合可能在节后发酵
  ├── 利率↑ → 重资产折现↓ → 🟡 电力 ETF/核电承压
  │   └── 电力=长久期资产，对利率最敏感（框架第 1 章）
  └── 利率↑ → 消费影响中性 → 🟡 海天
      └── 消费品利润稳定、近期现金流占比大，折现率变化影响小
```

**对比 6/17 作战计划的情景匹配**：

```
6/17 预测的情景 A（鹰派）→ 实际发生，且比预期更鹰：
  ✅ 美联储鹰派 → 实际：超级鹰派（废除前瞻指引+9票加息）
  ✅ 核电可能回调 → 实际：核电微涨（但节后可能滞后反应）
  ❌ 银行/消费继续承压 → 实际：银行逆势大涨！（加息预期→NIM逻辑驱动）
  ❌ 科技回调 → 实际：科技高位震荡但未大跌（国内政策对冲）
  
  → 关键偏离：银行板块的反应与预期相反——加息预期利好银行NIM，
    这个传导逻辑压倒了"资金抽血"的短期影响。框架的第2章（利率→银行
    两条路径）准确预测了这个结果，但6/17的作战计划没有充分应用。
```

**狂热阶段的操作原则**（霍华德·马克斯）：防御优先，不追高，等回调买。组合的防御属性在当前风格下承压，但这正是防御的意义——不在狂热中追科技，也不在恐慌中割防御。

### 💼 持仓分析——每只归因含框架引用

#### 农业银行（601288）— 连跌 3 日，但逻辑未破

| | |
|---|---|
| 收盘 | **6.53（-1.21%）** |
| 市值 | 5,877（成本 6,021）|
| 浮亏 | **-144（-2.39%）** |
| 距止损 | 5.82%（6.15）|

**归因**（框架：[[industry-banking-framework]]）：
- 银行板块整体 -1.14%，资金从防御→科技加速，陆家嘴论坛科技利好加剧了"高低切"
- 主力净流出 2.14 亿，散户接盘——典型的风格切换期筹码交换
- **银行的护城河（牌照+转换成本）没有变化**，NIM 1.28% 企稳趋势也没变（框架第 2 章）
- 连跌 3 日累计 -4.03%，但在 PB 0.81 的估值下，进一步下跌空间有限（估值方法论·农行 PB 分析）

**框架判断**：银行处于"弱周期复苏早期"（[[industry-cyclical-framework]] 第 4 章）。息差企稳+不良改善是慢变量，风格切换是快变量。快变量导致的下跌 = 周期问题不是结构问题。不操作。

#### 海天味业（603288）— 距止损最近，需最高警惕 ⚠️

| | |
|---|---|
| 收盘 | **33.32（-1.39%）** |
| 市值 | 6,664（成本 6,800）|
| 浮亏 | **-136（-2.00%）** |
| 距止损 | **6.12%（31.28）⚠️** |

**归因**（框架：[[industry-consumer-framework]]）：
- 调味品板块 -0.68%，消费板块全线下行，茅概念持续被抛售
- **但海天的护城河（品牌+渠道 6,700 家+规模 400 万吨）没有任何变化**（框架第 5 章）
- 合同负债 41.3 亿（↓5%）需要关注但不是危险信号（框架第 2.3 节）

**框架驱动的止损判断**（[[sell-discipline-framework]] 第 3.3 节决策树）：
```
海天连跌 3 日 → 是短期噪声还是结构问题？
  ├── 公司基本面恶化？→ 否（Q1 +10.96%，净利率 24.4%）
  ├── 行业周期问题？→ 是（风格切换+消费无催化）
  └── 公司结构问题？→ 否（品牌+渠道+品类扩张逻辑完整）

→ 结论：不卖。但必须盯紧 31.28（-8%）——触发就执行，不触发就不动。
```

**事前验尸**（[[universal-decision-framework]] 第 2 章）：
> "如果海天从 33 跌到 20（-40%），最可能的三个原因：①品类扩张失败，复合调味料增长降到个位数；②餐饮市场结构性萎缩；③经销商体系瓦解。"
> → 当前能看到任何一个的征兆吗？**不能。**
> → 所以下跌是估值压缩（PE 从 27→25 倍），不是基本面恶化。事前验尸告诉我：最大的风险不是公司出问题，是我在风格最不利的时候恐慌割肉。

**仓位超标问题**（框架第 4 章）：海天占 28%，纪律上限 20%。不是现在减——在浮亏时减仓是被剥夺超级反应驱动的错误决策（[[behavioral-finance-tools]] 第 1 章）。超标问题的解决时机是反弹后，不是在连续下跌时割肉。

#### 电力ETF（561560）— 政策催化对冲风格不利

| | |
|---|---|
| 收盘 | **1.372（-0.94%）** |
| 市值 | 3,567（成本 3,619）|
| 浮亏 | **-52（-1.44%）** |
| 距止损 | 6.63%（1.281）|

**归因**（框架：[[industry-power-framework]] + [[etf-investment-methodology]]）：
- 今日开盘高开 +0.29%（受发改委节能降碳政策催化），但随后低走
- **关键区分**（框架第 2 章）：电力**设备**板块暴涨 vs 电力ETF 偏传统电力（火电/水电/核电），前者受益于科技+电网投资逻辑，后者受风格切换拖累
- ETF 的成分结构决定了它在这种"科技 vs 传统"的分化行情中必然偏弱

**ETF 特有判断**（[[etf-investment-methodology]] 第 3 章）：
> ETF 下跌时不恐慌——只要行业景气度没有结构性问题。当前：火电复苏（煤价下行）+ 核电装机增长 + 风光扩张 = 景气度向上。ETF 越跌越买是合理的。

ETF 不需要像个股那样盯止损。只要煤价没有持续反弹 + 装机增长逻辑没变 → 继续持有。
