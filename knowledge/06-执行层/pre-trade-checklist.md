# 交易前自检卡片

> **每次买或卖之前，花1分钟跑完这7条。不跑不下单。**
> 这不是长篇大论，就是一张防手滑的清单。

---

## 🃏 正面：买入前七问

```
┌─────────────────────────────────────────┐
│         买入前自检卡片                    │
│         （下单之前花60秒）                │
├─────────────────────────────────────────┤
│                                          │
│  □ 1. 这公司怎么赚钱的？                  │
│     我能用一句话给朋友讲清楚吗？           │
│     → 讲不清 = 别买                      │
│                                          │
│  □ 2. 我是因为什么想买？                  │
│     □ 研究后觉得便宜                     │
│     □ 看了财报觉得基本面好               │
│     □ 朋友/网上有人说会涨（⚠️ 注意！）    │
│     □ 看到它一直涨，怕错过（⚠️ 危险！）   │
│     → 勾了后两项 = 别买，退一步冷静        │
│                                          │
│  □ 3. 买入价写下来了吗？                  │
│     我打算在_____元买入，为什么是这个价？  │
│     → 没理由的价格 = 赌博不是投资          │
│                                          │
│  □ 4. 止损线写下来了吗？                  │
│     买入价 × 0.92 = _____元（-8%止损）   │
│     → 现在看着这个数字，你能接受吗？       │
│     → 不能接受 = 仓位太大了，减量          │
│                                          │
│  □ 5. 仓位超限了吗？                      │
│     买完后，这只占我总资产____%？         │
│     → > 20% = 超限，别买这么多            │
│     买完后，这个行业占我总资产____%？      │
│     → > 40% = 超限，别买                  │
│                                          │
│  □ 6. 买完后如果明天就跌5%，我会怎样？     │
│     □ 没关系，我买的理由没变              │
│     □ 会很慌，后悔买了                    │
│     → 勾了第二个 = 仓位太重，减半          │
│                                          │
│  □ 7. 我现在情绪状态怎样？                │
│     □ 冷静，按计划行事                    │
│     □ 兴奋/焦虑/怕错过/想翻本             │
│     → 情绪驱动 ≠ 投资决策，延迟到明天再买  │
│                                          │
│  □ 8. 我判断错了，是因为什么？            │
│     □ 公司基本面判断错了                  │
│     □ 猜错了别人怎么想（选了没人选的）    │
│     → 前者=研究不够深，后者=不可避免      │
│     → 后者用仓位控制，不是不买的理由      │
│                                          │
│  □ 9. 我在第几层思考？                   │
│     □ Level-0：只看公司本身               │
│     □ Level-1：也看别人会怎么看           │
│     □ Level-2+：猜别人猜别人怎么想…       │
│     → 散户停在Level-1就够了              │
│     → 套太多层=过度自信，容易翻车         │
│                                          │
└─────────────────────────────────────────┘
```

## 🔄 背面：卖出前五问

```
┌─────────────────────────────────────────┐
│         卖出前自检卡片                    │
├─────────────────────────────────────────┤
│                                          │
│  □ 1. 我为什么想卖？                      │
│     □ 触发了止损线（纪律卖出 ✅）          │
│     □ 基本面恶化了（逻辑卖出 ✅）          │
│     □ 找到了更好的标的（调仓卖出 ✅）      │
│     □ 跌得太多，受不了了（⚠️ 恐慌卖出）    │
│     □ 涨了一点，想落袋为安（⚠️ 过早止盈） │
│     → 勾了后两项 = 等一下，先问自己第2问   │
│                                          │
│  □ 2. 我当初为什么买的？那个理由还在吗？    │
│     → 还在 = 不卖（价格波动不是卖的理由）  │
│     → 不在了 = 卖（逻辑破了必须走）        │
│                                          │
│  □ 3. 如果我没持有这只股票，今天会买吗？    │
│     → 会买 = 不卖（甚至考虑加仓）         │
│     → 不会买 = 卖（你的钱待错地方了）      │
│                                          │
│  □ 4. 现在卖是执行纪律还是情绪反应？       │
│     → 纪律（到了止损/止盈线）= 卖 ✅       │
│     → 情绪（恐慌/后悔/无聊）= 不卖 ❌      │
│                                          │
│  □ 5. 卖了之后这钱去哪？                  │
│     → 有明确的新标的 = 卖                  │
│     → 没想好 = 不卖（钱放着现金也行）      │
│                                          │
└─────────────────────────────────────────┘
```

## 📋 简化版——手机里存这个就行

如果你只想记一张卡片，就记这个：

```
交易前三问（每次下单前对自己说）：

1. 我为什么买/卖？这句话能写进决策日志吗？
2. -8%止损线写下来了吗？我能接受亏这么多吗？
3. 明天回头看，我会觉得今天是理性的还是冲动的？

→ 三个答案都是明确的 = 可以下单
→ 任何一个犹豫 = 今天不下单，明天再说
```

## ⚠️ 新手最容易踩的五个坑（卡片帮你防的）

| 坑 | 卡片帮你识别 | 对的姿势 |
|----|------------|---------|
| 追涨："它一直在涨，再不买就晚了" | 第2问：为什么买？——"怕错过"=危险 | 好公司也要等好价格，不追高 |
| 恐慌割肉："跌了好多，赶紧卖" | 卖出第2问：当初的理由还在吗？ | 跌不是卖的理由，基本面恶化才是 |
| 仓位太重："全买进去" | 第5问：仓位超限了吗？ | 单只不超过20%，永远留现金 |
| 听消息买："群里说这个要涨" | 第1问：能讲清楚怎么赚钱吗？ | 讲不清的不碰，不管谁说 |
| 赚钱就卖："涨了10%，赶紧跑" | 卖出第5问：卖了钱去哪？ | 好公司不要急着卖，让利润跑 |

## 📱 实用建议

1. **截图存手机**：把正面"买入七问"截图存相册，下单前翻出来看
2. **设交易密码**：把你的交易密码改成"止损了吗"的首字母（ZSLM+数字）
3. **延迟规则**：晚上想买 → 睡一觉，第二天早上再决定（90%的冲动会消失）
4. **决策日志**：每次下单后在日记里写一句话，半年后回头看

---

**Why:** 用户有一次买了长电科技第二天就卖了——典型的"试试手气"操作。新手最常见的亏损来源不是选错股，是情绪驱动交易。这张卡片把每次交易前该想的事压缩成60秒能跑完的7个问题，降低冲动交易的几率。

**How to apply:** 每次想买想卖时，打开这张卡片跑一遍。任何一个问题打了⚠️标记，今天的操作就取消。在[[investment-daily-log]]里记录每次交易后自检卡片的使用情况。
