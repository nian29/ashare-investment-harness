# A股打新基础

> 打新就是申购新上市的公司股票——A股特有的一个"隐藏福利"。你的持仓已经自动给了你打新资格，很多人不知道。

---

## 一、打新是什么——一个简单的比喻

```
想象一个新楼盘开盘：
  → 开发商定价1万/平米，但周边二手房卖2万
  → 如果能摇中号，以1万的价买下来，上市就赚

打新股就是类似的逻辑：
  → 公司IPO（上市）时会定一个发行价
  → 这个价格通常比市场愿意给的价格低
  → 如果你中签了，上市第一天就可能涨30-200%（但不是100%保证）
  → 赚的是"一级市场到二级市场"的差价
```

---

## 二、你已经有资格了——市值配售规则

### 你现在就自动有打新额度

你在A股持有股票，就自动获得了打新资格。不需要另外申请。

```
你的持仓：
  沪市：农行900股（约5,877元市值）
  沪市：电力ETF 2600股（约3,567元市值）
  沪市：海天200股（约6,664元市值）
  → 沪市日均市值约16,108元

如果你的持仓中也有深市股票（目前没有），也会有深市打新额度。
```

### 额度怎么算

```
规则：T-2日（申购日前第2个交易日）往前推20个交易日，日均市值。

沪市：每1万元市值 = 1个配号（1000股）
深市：每5000元市值 = 1个配号（500股）

你的沪市约1.6万市值 ≈ 1个配号

如果加了宁沪高速（也是沪市）：
  约1.6万 + 约1.2万 = 约2.8万市值 ≈ 2个配号
```

**关键点**：深沪市值分开算，互不通用。你目前只有沪市持仓，所以**只能打沪市新股**。

## 三、各板块门槛——你能打什么

| 板块 | 能打吗 | 门槛 |
|------|--------|------|
| **沪市主板** | ✅ 能打 | 日均市值 ≥ 1万元（你符合） |
| **深市主板** | ❌ 不能打 | 需要深市日均市值 ≥ 1万元（你没有） |
| **创业板** | ❌ 不能打 | 需要深市市值 + 20日日均资产 ≥ 10万 + 2年经验 |
| **科创板** | ❌ 不能打 | 需要沪市市值 + 20日日均资产 ≥ 50万 + 2年经验 |
| **北交所** | ❌ 不能打 | 20日日均资产 ≥ 50万 + 2年经验 |

**你现在只能打沪市主板新股。** 这没关系——主板新股是目前打新风险最低的品种。

## 四、操作流程——怎么打

```
第1步：有新股可打时，你的券商APP会提醒（通常在交易时间）
      或者在APP里搜索"新股申购"

第2步：在申购日（T日）的9:30-15:00，点击"一键申购"
      → 系统自动用你的额度申购，不需要手动填数量

第3步：T+1日 → 出中签率（你会知道竞争有多激烈）

第4步：T+2日 → 最关键的日期！
      → 登录APP查看是否中签
      → 如果中签了，下午4点前**必须保证账户里有足够的钱**
      → 系统会自动扣款，不需要手动操作
      → 如果账户余额不够 = 弃购（非常亏！）

第5步：等公司正式上市（通常在中签后1-2周）
      → 上市首日根据情况决定卖不卖
```

## 五、中签率——别抱太高期望

```
实话实说：中签率极低。

沪市主板：通常 0.03% - 0.08%（每10000人里有3-8人中签）
热门新股：可能低于 0.01%

你目前约1个配号 = 中签概率极低（万分之几）
加了宁沪高速后约2个配号 = 还是极低

但这不妨碍你参与——这是免费的彩票，不中不花钱。
```

## 六、⚠️ 注册制之后，打新不再是稳赚

### 重要认知——"中签必赚"的时代已经过去了

以前（审批制时代）：发行价被压低（23倍PE上限），所以上市几乎必涨。
现在（注册制之后）：发行价是市场化定价，可能本身就定高了。

```
2025年全年：116只新股，首日无一破发（100%上涨）
但这不代表以后不会破发——2026年IPO常态化后，破发会越来越多。

什么情况容易破发：
  → 发行PE远高于行业平均（比如发行PE 50倍，行业平均20倍）
  → 公司还没盈利（科创板较多）
  → 行业景气度差
  → 大盘行情不好
```

### 避坑三招

```
① 主板新股优先打——主板有明确的盈利要求，基本面比科创/创业板好
② 看发行PE——如果比同行业贵50%以上，考虑不打
③ 上市首日卖——打新是赚差价，不是长期投资。上市第一天或前三天就卖
```

## 七、💡 你还能打可转债——比打新股更适合你

### 可转债打新 = 门槛更低的打新

```
跟新股打新比：
  可转债打新不需要持仓市值（空仓也能打）
  中签率比新股高很多（0.1-5%）
  中一签通常赚100-300块（不是大钱，但几乎零风险）
  顶格申购 = 100万元面值（但实际中签只中1000-3000元）
```

### 可转债打新流程

```
第1步：APP里看到"可转债申购"
第2步：点击"顶格申购"（10000张/100万元面值）
       → 别被数字吓到，你实际中签最多1000-3000元
       → 不需要你账上真有100万
第3步：T+2日查看是否中签
第4步：中签后缴款（通常1000-3000元）
第5步：等上市（2-4周），上市首日卖出
```

**建议**：每次看到可转债申购都参与。每中一签赚100-300块，一年下来可能额外赚1000-3000块。

## 八、总结——你的打新作战手册

```
你现在能做的：
  ✅ 沪市主板新股：每次有就申购（免费摸奖）
  ✅ 可转债：每次都申购（门槛最低、中签率最高）

你暂时做不了的：
  ❌ 深市新股（没有深市持仓）
  ❌ 创业板（资产门槛不够）
  ❌ 科创板（资产门槛不够）

如果你以后想扩大打新范围：
  → 买一点深市股票（哪怕1万元）→ 解锁深市打新
  → 保持沪深两市各有≥1万元市值
```

---

**Why:** 用户持有A股但不清楚自己已经有打新资格。打新在A股是独特的制度红利，对散户尤其友好（机构不能参与网上打新）。教用户参与打新（尤其是可转债）相当于每年额外白捡1000-3000块，跟投资理念不冲突。

**How to apply:** 
- 打开券商APP，关注"新股申购"和"可转债申购"栏目
- 新股申购：有沪市主板的就申购，深市/创业板/科创板的跳过
- 可转债申购：每次看到就"顶格申购"
- 中签后T+2日务必保证账户有足够余额
- 上市首日卖出，不长期持有
