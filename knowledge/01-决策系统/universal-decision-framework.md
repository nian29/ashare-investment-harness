# 通用投资决策方法论

> 这是比任何具体框架都更底层的东西。不看行业、不选风格，只看决策质量。
> 巴菲特、索罗斯、西蒙斯——风格完全不同（价值/宏观/量化），但决策方法论惊人相似。

---

## 一、投资决策的第一性原理

所有投资决策的本质是同一个公式：

```
期望值 = 胜率 × 赔率 − 败率 × 损失

胜率：你判断正确的概率
赔率：如果判断正确，你能赚多少
败率：你判断错误的概率（= 1 − 胜率）
损失：如果判断错误，你会亏多少
```

**大多数人的错误**：只关注"我觉得能赚多少"（赔率），不考虑"我有多大把握"（胜率）和"错了会亏多少"（损失）。

### 你的持仓期望值对比

```
农行（PE 8x, PB 0.81, 股息 3.7%）：
  胜率：高（银行不会倒，低估值的均值回归概率高）
  赔率：低（涨到 PB 1.0 = +23%，天花板明显）
  损失：低（如果跌，有股息托底 + 国有大行有大而不倒保护）
  → 期望值：中等但稳健。高确定性、低赔率、低损失。

海天（PE 27x, 增速 11%）：
  胜率：中高（品牌+渠道垄断，但估值不便宜）
  赔率：中等（如果品类扩张成功 + 餐饮复苏，PE 回到 35 = +30%）
  损失：中等（如果增速放缓，PE 可能跌到 20 = -25%）
  → 期望值：中等。确定性不如农行，但赔率更高。

CPO 中际旭创（PE 114x, 增速 100%+）：
  胜率：低（技术路径、地缘政治、客户集中度）
  赔率：极高（如果一切顺利，可能再涨 50-100%）
  损失：极高（如果增速放缓，PE 从 114→30 = -70%）
  → 期望值：不确定。高赔率但低胜率、高损失。
```

**这个对比解释了你的防御偏好是对的**——农行和海天的期望值更好，不是因为它们"更会涨"，而是因为**损失可控**。

## 二、事前验尸——在做决策之前先想象它失败了

这是心理学上被验证最有效的决策工具之一，但投资圈很少人用。

### 怎么做

**在买入之前，写一段话**：

> "现在是 2028 年 6 月，我两年前买的 [标的] 亏损了 50%。回头看，最可能的原因是什么？"

不是泛泛地说"个股风险"或"宏观不好"，而是逼自己想出**3 个具体的、今天就能看到的失败原因**。

### 你的持仓事前验尸练习

**农行**：
> "2028 年回头看，农行亏了 30%。原因是：①中国进入零利率环境，NIM 从 1.28% 跌破 1.0%，盈利大幅缩水；②地方政府债务大规模违约，不良率从 1.27% 飙到 5%+，拨备不够用；③国有大行被迫承担政策性让利，分红被砍。"

**海天**：
> "2028 年回头看，海天亏了 40%。原因是：①复合调味料和零添加赛道涌入 5 家以上强敌，品类扩张失败；②餐饮市场结构性萎缩（预制菜+外卖改变调味品消耗模式），酱油销量见顶下滑；③线上渠道冲击传统经销商体系，6,700 家经销商出现批量退出。"

**核电**：
> "2028 年回头看，核电亏了 40%。原因是：①机制电价政策迟迟不落地，核电完全市场化，电价继续每年跌 5-8%；②在建 18 台机组因安全问题/政策审批延迟，投产节奏大幅慢于预期；③光伏+储电成本降到比核电更低，核电不再是基荷电源的最佳选择。"

**事前验尸的关键**：不是让你因为害怕而不买。而是让你**在买了之后，知道该盯什么信号**。如果某个失败原因开始出现征兆，你比其他持有者更早闻到味道。

## 三、贝叶斯更新——如何在持有期间调整判断

这是所有投资大佬都在用但很少明说的思维方法。

### 核心原理

```
初始信念（Prior）：买入时你对这家公司的判断
新证据（Evidence）：季报、行业数据、政策变化
更新信念（Posterior）：结合新证据后修正的判断

原则：不要因为一条新证据就彻底推翻初始信念（过度反应）
       但也不要在多条同向证据面前拒绝更新（确认偏误）
```

### 贝叶斯更新的三个要点

**1. 区分信号和噪声**

| 信号（应该更新你的判断） | 噪声（应该忽略） |
|------------------------|----------------|
| 连续两个季度的毛利率下降 | 单季度的天气影响 |
| 经销商数量和合同负债同向大幅减少 | 股价跌了 5% |
| 行业政策发生实质性改变 | 一篇唱空的媒体报道 |
| 竞争对手发布颠覆性产品 | 某分析师下调了目标价 |

**2. 给每条证据分配权重**

```
季报数据：权重高（自己看财报，不是看新闻）
行业数据：权重中（需要交叉验证）
公司公告：权重中高（但要注意公告的措辞变化）
媒体/研报：权重低（有利益冲突，需要验证）
股价波动：权重零（股价不是证据，股价是你需要解释的东西）
```

**3. 多条同向弱证据 = 一条强证据**

```
单条弱证据（如：单季度合同负债↓5%）→ 注意但不行动
两条同向弱证据（如：连续两季度合同负债↓）→ 减仓观察
三条以上（如：合同负债↓+经销商减少+毛利率↓）→ 很可能逻辑破了
```

### 海天当前实战

```
初始信念（2026.6）：海天是宽阔护城河的消费龙头，增速 10-12%，PE 25-30 合理

需要跟踪的新证据：
→ 2026Q2 季报（7-8 月）：
  → 合同负债是否继续减少？→ 如果是，权重 +1
  → 其他品类增速是否维持 14%+？→ 如果跌破 10%，权重 +1
  → 毛利率是否稳定？→ 如果下降，权重 +1

如果三条同向恶化 → 更新信念：品类扩张可能不如预期
                      → 下调合理 PE 区间到 22-27 倍
                      → 如果当前 PE 高于新区间，减仓
```

## 四、决策日志——把运气和能力分开

投资最大的敌人不是亏损，是**分不清自己赚钱是因为判断对了还是运气好**。

### 决策日志模板

不是复盘（复盘是交易后做的），决策日志是**买入/卖出时立刻写的**：

```
━━━━━━━━━━━━━━━━━━━━━━━━
决策日志 | ID: ____ | 日期: ____
━━━━━━━━━━━━━━━━━━━━━━━━

操作：□ 买入 / □ 卖出 / □ 加仓 / □ 减仓 / □ 不动（主动选择）
标的：____

1. 这个决策基于什么证据？
   _________________________________

2. 如果我是错的，最可能的原因是什么？（事前验尸）
   _________________________________

3. 三个月后回头看，我需要看到什么信号来证明我是对的？
   _________________________________

4. 什么信号会让我改变这个判断？
   _________________________________

5. 这个决策的质量
   □ 有充分研究 + 逻辑清晰 + 情绪冷静
   □ 有研究但不够深入
   □ 主要靠直觉
   □ 情绪驱动（FOMO/恐慌/别人推荐）
```

### 为什么必须写决策日志

```
三个月后回头检查第 3 问和第 4 问：
→ 如果证明你正确的信号出现了 → 逻辑在验证中，可以继续持有或加仓
→ 如果让你改变判断的信号出现了 → 逻辑破了，执行卖出
→ 如果什么都没出现 → 再等一个月，但要设时间止损

一年后统计第 5 问：
→ "情绪驱动"类决策的平均收益 vs "充分研究"类决策的平均收益
→ 你会发现自己做了多少不必要的情緒交易
→ 数据比任何人的劝告都更有说服力
```

## 五、OODA 循环——持续迭代的操作系统

OODA 是军事术语（Observe-Orient-Decide-Act），被投资界广泛采用。它描述的是**持续决策的循环**，不是一次性的分析。

```
    ┌──────────────────────────────────┐
    │                                    │
    ▼                                    │
Observe（观察）                           │
→ 市场发生了什么？                         │
→ 我的持仓发生了什么？                      │
→ 有什么新信息？                           │
    │                                    │
    ▼                                    │
Orient（定向）                            │
→ 这些新信息意味着什么？                    │
→ 它改变了我对"合理价值"的判断吗？           │
→ 用贝叶斯更新：我的信念应该调多少？         │
    │                                    │
    ▼                                    │
Decide（决策）                            │
→ 买 / 卖 / 持有 / 加仓 / 减仓？           │
→ 期望值够不够？事前验尸过了吗？             │
    │                                    │
    ▼                                    │
Act（执行）                               │
→ 按纪律执行（不盘中临时改）                │
→ 写决策日志                               │
    │                                    │
    └──────────────────────────────────────┘
```

**你的投资手册已经定义了这个循环**（每日检查 = Observe，综合判断 = Orient + Decide，交易纪律 = Act），但缺少**Orient（定向）这一步的深度**——即"新信息来了，我应该更新多少旧判断"。这正是贝叶斯更新要解决的。

## 六、不同投资风格在这个通用框架下的映射

这套通用方法对任何风格都适用，只是输入变量不同：

| 通用方法 | 价值投资（你） | 成长投资 | 宏观对冲 | 量化 |
|---------|-------------|---------|---------|------|
| **期望值** | 低 PE+稳增长 = 高概率中等回报 | 高增速 = 低概率极高回报 | 宏观方向正确 = 中等概率高回报 | 统计优势 = 52-55% 微弱概率优势 × 极高频次 |
| **事前验尸** | 护城河瓦解、估值中枢下移 | 增速放缓、竞争加剧 | 宏观判断方向反了 | 因子失效、过拟合 |
| **贝叶斯更新** | 季报+行业数据 | 用户增长、市占率趋势 | 利率预期、经济数据 | 回测 vs 实盘表现 |
| **决策日志** | 关键（区分运气和判断） | 更关键（成长股的波动让运气占比更大） | 关键 | 极关键（高频决策需要日志来发现系统性偏误） |

## 八、最小可行操作——如果你只做一件事

如果只能从这篇里拿走一条，**拿事前验尸**。

买入任何东西之前，用 3 分钟写一段话——想象三年后这笔投资亏了 50%，列举 3 个最可能的具体原因。写下来，存着。

一年后翻出来看。**你会发现你当时已经隐约知道哪里会出问题，只是选择性忽略了。** 这就是事前验尸的力量——它强迫你面对你已经在潜意识里感到不安的东西。

---

**Why:** 用户有 21 篇具体领域的投资记忆，但缺少最底层的"决策操作系统"——期望值思维、事前验尸、贝叶斯更新、决策日志、OODA 循环。这些方法不挑风格、不挑行业，是任何投资决策都应该遵守的通用原则。

**How to apply:** 每次做买卖决策前跑期望值（三问：胜率？赔率？损失？）→ 事前验尸（想象三年后失败了为什么）→ 写决策日志（半年后回看）。每次收到新信息时用贝叶斯更新（信号 vs 噪声？权重多少？累加到什么程度才行动？）。
