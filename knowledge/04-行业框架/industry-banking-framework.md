# 银行业分析框架

> 配合 [[investment-handbook]] 维度一和维度二使用。
> 银行是三张表最特殊的行业——"原材料"是存款，"产品"是贷款，"价格"是利率。用普通企业的逻辑看银行会完全走偏。

---

## 一、银行的生意本质

用最简单的话说：

```
银行赚的钱 = 贷款收的利息 − 存款付的利息 − 坏账损失 + 中间业务收入
           = 净利息收入 + 非息收入 − 信用减值损失
```

银行的"原材料"是存款（负债端），"产品"是贷款（资产端），"价格差"是净息差。

**这意味着银行的利润有三个驱动变量**：
1. 量：贷款规模能不能增长？（受制于资本充足率）
2. 价：净息差是扩大还是收窄？（受制于利率环境和竞争格局）
3. 质：放出去的贷款会不会变成坏账？（资产质量）

---

## 二、核心指标四件套（每次必看）

### 指标 1：净息差（NIM）

```
净息差 = 生息资产收益率 − 计息负债成本率
```

**净息差是银行盈利能力的核心。** 一个简便的判断：

| NIM 水平 | 含义 |
|----------|------|
| > 2.5% | 较高（通常是零售型银行或高息差环境） |
| 1.8-2.5% | 正常 |
| 1.3-1.8% | 偏低（国有大行当前水平） |
| < 1.3% | 危险——息差接近盈亏平衡线 |

**农行 2025 实战**：
```
净息差：1.28%（已处于历史最低水平）
→ 生息资产收益率：2.69%（↓43bp，贷款利率在快速下行）
→ 计息负债成本率：1.53%（↓32bp，存款成本也在降）
→ 净利差：1.16%

关键判断：NIM 1.28% 能不能稳住？
→ 有利因素：存款成本率在降（1.34%，↓29bp），县域低成本存款是农行的结构性优势
→ 不利因素：贷款收益率在快速下行（-53bp），且贷款利率短期内看不到止跌信号
→ Q4 单季降幅收窄至 14bp → 企稳迹象，但方向确认需要看 2026Q1-Q2
```

**一个灵魂问题**：净息差降到多低，银行就不赚钱了？

```
粗略计算：
农行管理费率 ≈ 1.3%（成本收入比）
信用成本 ≈ 0.5-0.8%（不良生成率 × (1−回收率)）

息差覆盖 = NIM − 管理费率 − 信用成本
         = 1.28% − 1.3% − 0.7% ≈ −0.72%

→ 如果只看净利息收入，息差已经覆盖不了成本！
→ 所以非息收入（1,557 亿，+19.9%）现在是农行的核心增长引擎
→ 银行业的未来：从"吃利差"转向"吃服务费"
```

### 指标 2：不良贷款率 + 关注类贷款迁徙率

不良率是静态的，**关注类贷款迁徙率是动态的**——它告诉你"下一批不良从哪来"。

```
不良贷款迁徙路线：
正常 → 关注 → 次级 → 可疑 → 损失
          ↑                        ↑
      迁徙率告诉你          不良贷款 = 次级+可疑+损失
      多少关注在变成次级
```

**农行 2025 实战**：
```
不良率 1.27%（↓3bp）→ 静态改善
关注率 1.39%（↓1bp）→ 不良的"蓄水池"没扩大
不良生成率 0.57%（上年 0.60%）→ 新增不良在减少

→ 三个指标同向改善 → 资产质量真实好转
→ 但零售不良率 1.34%（+16bp）→ 房贷和经营贷承压，分化在加剧
→ 对公不良 1.37%（↓8bp）→ 对公改善，零售恶化
```

**判断不良的核心问题**：
```
逾期 90 天以上 / 不良贷款 = 71.95%
→ 逾期 90 天以上贷款 < 不良贷款
→ 说明农行把不良认定得更严——有些逾期没到 90 天的也提前划入不良了
→ 这是审慎的信号，不良率 1.27% 是可信的

如果这个比率 > 100%（即逾期 90+ 的贷款比不良多），说明银行在故意少报不良。
```

### 指标 3：拨备覆盖率

```
拨备覆盖率 = 贷款损失准备 / 不良贷款余额

这相当于银行给坏账准备的"安全气囊"：
- 200% = 每 1 元不良贷款准备了 2 元来弥补损失
- 监管要求 150%（农行这类大行要求更高）
```

**农行 2025 实战**：
```
拨备覆盖率 292.55%
→ 极高水平，远超监管要求
→ 相当于"藏了"大量利润——未来即使不良上升，也不需要多提拨备
→ 拨备下降 → 利润释放（管理层可以用这个来平滑利润）
→ 拨备上升 → 利润隐藏（管理层对未来悲观，多计提做储备）

当前 292% → 有向下调节的空间 → 利润有"安全垫"
```

**拨贷比** = 拨备余额 / 总贷款 = 3.71%（农行 2025）。拨备覆盖率 × 不良率 = 拨贷比。两个指标合在一起看才有意义。

### 指标 4：资本充足率

```
资本充足率 = 资本净额 / 风险加权资产

银行是"资本消耗型"生意——放贷需要占用资本。资本充足率直接决定了贷款增速的上限。
```

**农行 2025 实战**：
```
核心一级资本充足率：10.96%
→ 高于监管要求（8.5%，含缓冲），但并不宽裕
→ 每年贷款增长 ~8% → 每年消耗约 0.8 个百分点的资本
→ 资本内生速度（ROE × (1−分红率)）= 10.16% × (1−32%) ≈ 6.9%
→ 内生资本增长基本够用，但没太多富余
→ 如果未来要加大分红，资本充足率会承压
```

## 三、银行的 ROE 拆解——理解"为什么 ROE 只有 10%"

```
ROE = 净利润 / 净资产 = 10.16%（农行 2025）

杜邦拆解：
ROE = (净利润/总收入) × (总收入/总资产) × (总资产/净资产)
    = 净利率 × 资产周转率 × 权益乘数

农行：
净利率 ≈ 40%         → 每赚 100 块收入，40 块是利润（看起来很高）
资产周转率 ≈ 1.5%    → 但 1 块钱资产只能产生 1.5 分钱收入（极低！）
权益乘数 ≈ 16x       → 靠 16 倍杠杆把极低的周转率拉回来的

→ 40% × 1.5% × 16 ≈ 10%
→ 银行的高 ROE 靠的是杠杆，不是经营效率
→ 所以银行是高风险行业——杠杆越高，风险越大
→ 这也是为什么银行股 PE 永远低：因为有杠杆，利润的波动性被放大了
```

**对比海天**：
```
海天 ROE 19.59% ≈ 净利率 24.4% × 周转率 × 杠杆 1.2x
→ 不需要杠杆，靠的是净利率和周转率
→ 这种 ROE 质量远高于银行的 ROE
→ 所以海天值 27 倍 PE，农行只值 8 倍 PE——不是因为谁"更好"，是因为利润质量不同
```

## 四、投资银行股的三条路径

### 路径 A：吃股息（你现在的策略）

适合标的：农行、建行、中行等国有大行

**核心逻辑**：不赌增长，吃分红。前提是——分红可持续、不会砍。

**关键检验**：
```
□ 股息率 > 3.5%？
□ 分红比例稳定或上升？（农行 32%，有提升空间）
□ 资本充足率足够支撑分红？（内生资本增速 ≥ 贷款增速 + 分红需求）
□ 不良率没有恶化趋势？（如果不良率上升，拨备会吃掉利润 → 分红可能被砍）
```

**农行当前**：股息率 ~3.7%，PB 0.81 → 典型吃股息标的。价格越低，股息率越高，逻辑越强。

### 路径 B：赌资产质量反转

适合标的：股价被资产质量担忧压得很低的银行

**核心逻辑**：市场认为不良率高、拨备不够 → PB 被压到极低 → 但实际不良没有想象那么差 → 拨备释放 + PB 修复 = 双击。

这种机会不常有，一般出现在极端悲观时（比如市场认为地方政府债务会让银行不良率飙升）。

### 路径 C：赌增长

适合标的：招商银行、宁波银行等零售/财富管理强的银行

核心逻辑：零售银行有更强的定价权（信用卡利率高、存款成本低）、非息收入占比高、ROE 高于大行。但估值也更高（招行 PB ~1.0 vs 农行 0.81）。

## 五、农行 2025 综合评估

用晨星护城河框架对照：

```
□ 护城河定性：牌照（有效规模）+ 转换成本
   → 银行牌照是稀缺资源，新进入者几乎不可能
   → 企业换银行 = 所有贷款/结算/代发工资要重签，转换成本极高

□ 护城河宽度：宽阔
   → 四大行基本不可替代，央行最后贷款人身份决定大而不倒

□ 护城河趋势：稳定偏正面
   → 县域存款成本优势（1.28%）是全行最低的——别人拿不到这么便宜的钱
   → 但息差下行压力仍在，需要关注

□ 假护城河排查：
   ⚠️ ROE 10% 不是高——靠杠杆撑起来的，不要被 40% 净利率迷惑

□ 进化分类：传统护城河（壁垒强但增长有限，回报靠分红）
```

**投资逻辑一句话**：农行不是买增长的，是买分红 + 买资产质量改善 + 买 PB 修复的。

## 六、分析银行股的 15 分钟检查清单

```
⏱ 0-5 分钟：盈利引擎
  → 净息差是多少？趋势？（上升/下降/企稳？）
  → 非息收入增速？（> 利息收入增速 = 转型中）

⏱ 5-10 分钟：资产质量
  → 不良率 + 关注率 + 不良生成率（三个一起看）
  → 逾期 90+/不良（< 100% = 认定严格）
  → 拨备覆盖率（> 200% = 安全垫充裕）

⏱ 10-15 分钟：资本与回报
  → 资本充足率 → 能支撑多少贷款增长？
  → ROE 杜邦拆解 → 利润质量
  → 股息率 + 分红比例 → 我能拿到多少？
```

---

**Why:** 用户持有农行，但缺少银行业的特化分析框架——用普通企业的指标看银行会得出错误结论。本手册以农行 2025 年报为案例，拆解银行的核心盈利逻辑。

**How to apply:** 每次评估银行股时先跑四件套（NIM+不良+拨备+资本），再跑 ROE 杜邦拆解，最后对照护城河框架。
