# 交通运输/基础设施行业分析框架

> 这个框架帮你分析"收过路费"类生意——高速公路、港口、铁路、机场。它们的共同特点是：**前期砸大钱建基础设施，建好后坐地收钱**。生意模式好懂，关键是要知道盯什么数字。

---

## 一、这个行业为什么适合吃分红的人

四个字：**现金奶牛**。

| 特征 | 什么意思（说人话） |
|------|-------------------|
| 收入是现金 | 过路费、港口费、火车票——都是现收的，不赊账 |
| 成本大都是纸面的 | 最大成本是折旧（建路建港时花的钱摊到每年），但折旧不掏现金 |
| 经营现金流 > 净利润 | 账面利润可能是10亿，但实际收回的现金可能是15亿 |
| 赚来的现金大部分分掉 | 因为不需要大量再投入（除非改扩建），所以分红比例高 |

**一句话**：这类公司账面看着利润一般，但兜里现金很充裕——所以能稳定地给你分红。

## 二、高速公路——最核心的细分行业

### 2.1 怎么赚钱的

```
收入 = 车流量 × 每辆车收多少钱 × 跑了多长距离

车流量：多少车经过（看经济和汽车保有量）
收费标准：政府定的，一般小车0.4-0.6元/公里（公司说了不算）
里程：路有多长
```

**关键认知**：收费标准是政府定的，公司调不了价。所以**高速公路的利润增长主要靠车流量增长**，不是靠涨价。

### 2.2 必须盯住的四个核心指标

| 指标 | 去哪看 | 怎么判断好坏 |
|------|--------|------------|
| **日均车流量** | 公司月度/季度经营数据公告 | ↑ 车多了 = 收入增。连续两个季度降就要警惕 |
| **客货比** | 年报/调研纪要 | 客车多 = 稳（出行需求刚性）；货车多 = 跟经济周期走 |
| **单公里通行费收入** | 自己算：年收入 ÷ 总里程 | 越高说明路段越优质。东部发达地区 > 中西部 |
| **毛利率** | 年报利润表 | 通常50-60%，稳定就好。骤降说明养护成本暴增或车少了 |

### 2.3 行业特有的两大利好逻辑

**利好1：改扩建——"老路翻新、重新计时"**

这是高速公路行业最重要的一句话：**路扩建了，收费年限重新算。**

- 原本一条路收费到2032年到期（还剩6年），价值越来越低
- 如果进行改扩建（四车道变八车道），完工后收费年限**从完工之日起重新核定25年**
- 相当于：本来还剩6年收钱的资产，突然变成还能收25年——价值直接翻几倍

实际案例（宁沪高速自己的历史）：
- 2004-2006年沪宁高速改扩建，完工后车流量提升了56%
- 改扩建期间车流量会下降（施工影响通行），收入短期承压
- 所以改扩建 = **短期痛苦、长期受益**

**利好2：收费管理条例修订——"政策松绑"**

- 现行规定：经营性公路收费最长25年
- 正在修订：可能允许东部地区收费超过30年
- 如果落地：那些快要到期的路产突然"续命"，公司价值重估

### 2.4 行业特有的三大风险

| 风险 | 什么意思 | 怎么跟踪 |
|------|---------|---------|
| **收费权到期** | 路不是永久的，到期要还给政府 | 查每条路的到期年份，最近的越危险 |
| **分流效应** | 旁边新修一条路，车被分走 | 关注地方交通规划，新建平行路段 |
| **改扩建资本开支** | 扩建期间要花大钱，短期压利润 | 看每年的资本开支计划，高峰期的公司分红可能暂时减少 |

### 2.5 怎么给高速公路股估值

```
不要只看 PE！PE 会骗你。

原因：高速公路利润里有大量折旧（纸面成本），
      PE = 股价 ÷ 净利润，而净利润被折旧压低了。
      → PE 看起来偏高，但实际上公司现金流很好。

更靠谱的指标：
① 股息率：每股分红 ÷ 股价。> 4% = 有吸引力
② 企业价值/EBITDA：比PE更准，因为EBITDA去掉了折旧的影响
③ 市净率（PB）：修一条路花多少钱 vs 市值多少，< 1.5倍算不贵
```

**实用速查**：高速公路股合理PE通常在10-15倍（比银行和建筑高，因为现金流更好）。宁沪当前13倍，属于合理区间。

## 三、高速公路选股四步法

```
第一步：看路产位置
  → 路在长三角/珠三角/京津冀？→ 车流量有保障 ✅
  → 路在中西部偏远地区？→ 车流量不稳定 ⚠️

第二步：看收费到期时间
  → 核心路产到期 > 10年？→ 暂时安全 ✅
  → 核心路产到期 < 5年 + 没有改扩建计划？→ 注意 ⚠️

第三步：看改扩建管线
  → 有明确的改扩建计划 + 完工后重新核定年限？→ 成长逻辑清晰 ✅
  → 没有改扩建计划 + 路产快到期？→ 价值会越来越低 ⚠️

第四步：看分红承诺
  → 明确承诺分红比例 > 60%？→ 分红有保障 ✅
  → 分红随利润波动 + 没有承诺？→ 分红不确定性高 ⚠️
```

## 四、宁沪高速（600377）实战拆解

用上面的框架跑一遍刚推荐的宁沪高速：

### 路产位置：✅ 优质
- 核心路产沪宁高速连接上海和南京，长三角经济核心区
- 中国最繁忙的高速公路之一
- 单公里收入在全国排前列

### 收费到期时间：⚠️ 关注但不急
```
沪宁高速江苏段 → 2032年6月到期（还剩6年）← 这是最大的隐忧
锡宜高速 → 已改扩建，待完工后重新核定到2056年
广靖高速 → 改扩建中，2027年完工后重新核定
锡澄高速 → 改扩建已启动
```

**判断**：沪宁高速还有6年，而且公司已经在做改扩建的"优化研究"。如果未来几年启动改扩建 + 重新核定年限，这个最大的隐忧就消除了。

### 改扩建管线：✅ 催化剂充足
- 广靖、锡澄正在改扩建（预计2027年完工）
- 锡宜南段已完工（2025年12月通车）
- 还有丹金高速、锡太高速新建项目（2028年通车）
- **短期阵痛**：资本开支高峰期（2026年计划117亿），短期压制自由现金流
- **长期利好**：每条路完工后重新核定25年，相当于"资产续命"

### 分红：✅ 硬气
- 2025年利润降了7%，但分红一分没少（每股0.49元维持不变）
- 公司明确表态"分红金额不回调"
- 上市以来分红融资比50倍（分红412亿，融资8亿），行业第一

### 当前核心矛盾

```
利好                           利空
─────────────────────────────────────
车流量稳定（长三角经济强）      沪宁高速2032年到期（还剩6年）
改扩建管线充足（完工就续命）    资本开支高峰期（短期压现金流）
分红硬气（不降分红）            沿江高速分流效应退坡
收费条例修订在望（政策催化）    改扩建有不确定性
```

**总体判断**：好资产 + 合理价格 + 短期有阵痛（资本开支）+ 长期有催化（改扩建续命 + 政策松绑）。适合愿意拿分红、同时能忍受短期波动的投资者。

## 五、港口——另一种"收过路费"生意

### 怎么赚钱的
```
收入 = 货物吞吐量 × 单吨收费
→ 货多了就赚得多，跟进出口贸易高度相关
```

### 关键指标
| 指标 | 含义 |
|------|------|
| 货物吞吐量 | 一共装卸了多少吨货 |
| 集装箱吞吐量 | 高端货（电子产品等）用集装箱，利润率更高 |
| 单吨收入 | 每吨货收多少钱，看港口竞争力 |

### 投资逻辑
- **优点**：也是现金奶牛、区域垄断（一个地区通常只有一个大港）
- **缺点**：比高速公路更依赖宏观经济——贸易战、全球经济衰退直接冲击吞吐量
- **典型标的**：上港集团（长三角）、宁波港、唐山港（股息率高）

---

## 六、铁路——客运 vs 货运，差别很大

### 客运铁路（如京沪高铁）
- 收入 = 客流量 × 票价
- 票价也是政府管控的
- 优点是客流量极其稳定（京沪线不愁没人坐）
- 缺点是增长有限（票价不能随便涨，线路满了加不了车）

### 货运铁路（如大秦铁路）
- 收入 = 运煤量 × 运费
- 跟煤炭行业深度绑定
- 周期性比客运铁路强很多

---

## 七、机场——免税店才是主角

### 不靠飞机起降赚钱
```
机场收入分两块：
① 航空性收入：飞机起降费（政府定价，赚不了多少）
② 非航空收入：免税店租金、商铺租金、广告（这才是利润大头）

上海机场的核心资产不是跑道，是免税店的租金分成。
```

### 核心指标
- 旅客吞吐量（尤其是国际旅客——免税店消费主力）
- 非航空收入占比（越高说明赚钱能力越强）

## 八、总结——这个行业怎么投

```
如果只想稳稳吃分红：
  → 高速公路 > 港口 > 客运铁路 > 货运铁路 > 机场
  → 首选长三角/珠三角的优质高速路段

如果想赌一把增长：
  → 看有改扩建管线的公司（完工后量价齐升）
  → 或者有收费管理条例修订落地的政策催化

如果想分散风险：
  → 高速公路ETF（如果未来有的话）
  → 或者宁沪+粤高速+招商公路各买一点，分散单条路的风险
```
