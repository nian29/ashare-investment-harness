# 医药行业分析框架

> 医药是 A 股仅次于消费的第二大"好赛道"——高毛利、强专利护城河、刚需。但也是最复杂的行业之一：研发管线决定未来、政策（集采）可以摧毁现有利润。

---

## 一、医药行业的生意本质

医药不是一个行业，是**两个行业拼在一起的**：

| | 创新药 | 仿制药 |
|------|--------|--------|
| **护城河** | 专利（20 年独占）+ 临床数据 | 几乎没有（专利过期后大家都能做） |
| **毛利率** | 80-95% | 30-60%（集采后更低） |
| **增长逻辑** | 研发管线 → 新药获批 → 放量 | 成本优势 + 渠道覆盖 |
| **最大风险** | 研发失败（90% 失败率） | 集采降价（-50~-90%） |
| **估值方法** | DCF 折现研发管线（PE 没用） | PE + PEG（类似消费品） |
| **A 股代表** | 恒瑞医药、百济神州、信达 | 华海药业、人福医药 |

---

## 二、创新药的核心逻辑——研发管线估值

创新药公司不能用 PE 估值，因为：
- 收入来自若干年后才获批的药（现在可能还在做临床）
- 利润可以被研发费用大幅压低（研发投入越多 → 当期利润越低 → 但未来管线越强）

### 2.1 研发管线

```
一条药物从实验室到上市需要 10-15 年：

临床前 → 临床 I 期 → 临床 II 期 → 临床 III 期 → 审批 → 上市
  ↑                    ↑                      ↑
 成功概率 ~10%      成功概率 ~30%          成功概率 ~60%
```

**管线的价值 = 峰值销售额 × 成功概率 × 折现**

```
例子（简化）：
某抗癌药临床 III 期，预计峰值销售额 30 亿/年
成功概率 60%
折现后价值 ≈ 30 亿 × 60% × 合适的倍数

→ 创新药公司 = 所有管线价值的加总
→ 管线越多、阶段越靠后、适应症越广 = 价值越高
```

### 2.2 创新药的三个关键问题

**问题 1：靶点是不是太拥挤？**

```
PD-1（抗癌免疫治疗）：全球 100+ 家公司在做 → 太拥挤
→ 后来者很难拿到理想份额
→ 恒瑞的 PD-1（卡瑞利珠单抗）是前五获批的，但竞争格局在恶化
```

**问题 2：适应症空间有多大？**

```
一个药可能有多个"适应症"（批准的用途）
→ 恒瑞的某药物获批肺癌（大市场）vs 某罕见病（小市场）
→ 适应症越多 → 峰值销售额越高
→ 适应症从末线（晚期病人）向前线（早期病人）推进 → 患者人群指数级增长
```

**问题 3：全球化验证了吗？**

```
中国创新药估值的天花板：国内市场
→ 出海（FDA 批准 + 海外授权）是估值的跳变点
→ 恒瑞 FDA 评审（手册里提的 7 月）就是这个逻辑
→ 如果 FDA 批准 → 海外市场打开 → 估值逻辑重写
→ 如果 FDA 拒绝 → 出海受阻 → 估值回调
```

## 三、仿制药的核心逻辑——集采是最大的变量

仿制药的逻辑简单：专利过期后，大家都能做，拼成本和规模。

**集采（带量采购）改变了游戏规则**：

```
集采前：药企通过销售代表推广 → 价格高 → 毛利率高
集采后：政府统一招标 → 最低价中标 → 价格跌 50-90%

→ 仿制药从"品牌消费品"变成了"制造业"
→ 毛利率从 60% 降到 30-40% 甚至更低
→ 以前值 25-30 倍 PE，现在只值 12-18 倍 PE
```

**恒瑞的转型**就是从仿制药转向创新药——不再靠仿制药赚钱，而是靠创新药（有专利保护、不怕集采）。

## 四、恒瑞医药——你关注过的标的实战分析

### 恒瑞当前状况

```
2025 年关键变化：创新药收入占比首超 60%（里程碑事件）

这意味着：
→ 恒瑞已经从"仿制药企"变成了"创新药企"
→ 集采对它的影响在边际递减（仿制药占比越来越小）
→ 估值应该从仿制药 PE（15-20x）向创新药估值切换
→ 但创新药估值体系更复杂（看管线，不是看 PE）
```

### 恒瑞管线概览

```
核心管线（2026 年估值的主要支撑）：
□ 卡瑞利珠单抗（PD-1）- 已上市，核心品种
□ 多个联合用药方案 - 临床 II-III 期
□ 多款创新药在 FDA 审评中 - 2026 年 7 月关键节点
□ 差异化布局（不跟 PD-1 内卷，扩展新靶点）
```

### 恒瑞的关键催化剂和风险

```
✅ 催化剂：
  → FDA 批准（2026.7）→ 出海逻辑兑现 → 估值跳变
  → 创新药收入占比继续提升 → 估值向创新药切换
  → 新适应症获批 → 现有药物患者人群扩大

❌ 风险：
  → FDA 拒绝 → 出海受阻 → 估值逻辑受损
  → PD-1 竞争继续恶化 → 核心品种增长放缓
  → 国内新一轮医保谈判降价 → 已在售创新药被压价
  → 集采对新批仿制药仍有冲击
```

### 恒瑞的估值

```
不能用单一 PE 看恒瑞：
→ 创新药部分（60%+ 收入）：看管线 DCF
→ 仿制药部分（< 40% 收入）：PE 15-20x

综合判断：恒瑞的合理估值取决于创新药管线进展
→ FDA 批准 + 新药放量 → 估值上行
→ 管线受挫 + 竞争恶化 → 估值下行

52 周低点附近是市场最悲观的时候（像马克斯说的"在别人恐惧时贪婪"）
但跌到低点 ≠ 一定反弹——需要确认 FDA 评审和管线进展后再动手
```

## 五、医药行业分析检查清单

```
⏱ 0-5 分钟：定位公司类型
  → 创新药还是仿制药？（恒瑞是"创新药为主、仿制药为辅"）
  → 核心品种是什么？专利什么时候到期？
  → 研发管线在什么阶段？（临床 I/II/III？）

⏱ 5-10 分钟：政策风险评估
  → 集采影响哪些品种？占比多少？
  → FDA/海外评审关键事件时间表？

⏱ 10-15 分钟：估值
  → 创新药部分：管线估值（峰值销售×概率×折现）
  → 仿制药部分：PE 12-18x（集采后）
  → 混合型公司 = 两部分分开估值再相加
```

---

**Why:** 用户关注过恒瑞医药但缺少医药行业的分析框架。医药行业是 A 股第二大"好赛道"，但也是最复杂的——创新药和仿制药是完全不同的两种生意，需要不同的分析工具。

**How to apply:** 先判断公司是创新药还是仿制药（恒瑞是创新药为主），然后用对应的估值方法。创新药看管线（不只看 PE），仿制药看集采影响。恒瑞的 FDA 评审是核心催化剂。
