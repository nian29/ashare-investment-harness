# 消费/食品行业分析框架

> 配合 [[investment-handbook]] 维度一和维度二使用。
> 消费品是 A 股财务最干净、逻辑最直觉的板块——但干净不等于没有盲区。消费股的陷阱往往不在数字里，在渠道和竞争格局里。

---

## 一、消费品的生意本质

```
利润 = 销量 × 单价 × 毛利率 − 销售费用 − 管理费用
     = 量 × 价 × 利润率
```

**消费品增长的三引擎**：

| 引擎 | 含义 | 可持续性 |
|------|------|---------|
| **量** | 渗透率提升——更多人买，或者每个人买更多 | 最好的增长（可持续） |
| **价** | 提价能力——涨价后销量不跌 | 次好的增长（但天花板有限） |
| **品类扩张** | 酱油 → 蚝油 → 调味酱 → 复合调味料 | 最关键的长期增长来源 |

**消费股的核心护城河**——就是品牌能不能让你提价而客户不会跑。检验方法很简单：

> 海天酱油提价 5%，消费者会不会换品牌？
> 如果不会 → 有护城河（品牌定价权）
> 如果会 → 没有护城河（同质化产品）

## 二、消费股核心指标的真正含义

### 2.1 毛利率——产品有没有议价权

```
毛利率高 = 产品有差异化 ≠ 有护城河
毛利率稳定或上升 = 竞争格局好
毛利率趋势性下降 = 竞争在加剧（对手在压价）
```

**海天 2025**：

```
毛利率 40.22%（+3.22pct）
→ 上升 3.22 个百分点，主要原因是原料成本下降 + 灯塔工厂降本
→ 这不是提价带来的，是成本端改善
→ 问：如果原料成本回升到 2023 年水平，毛利率会跌回去吗？
→ 灯塔工厂带来的效率提升是结构性的（不会因为原料涨就消失）
   → 但原料成本改善是周期性的（可能反复）
→ 所以不能线性外推毛利率 40%+
```

### 2.2 净利率——经营效率的刻度

```
海天 2025 净利率 24.40%（+0.77pct）
→ A 股食品行业最高水平之一
→ 竞争对手（中炬、千禾）净利率在 8-15%
→ 差距来自：海天规模更大 → 采购成本更低 → 费用率更低
→ 这是一个**自我强化的护城河**：规模越大 → 成本越低 → 利润越多 → 更有钱做品牌和渠道
```

### 2.3 合同负债（预收款）——经销商信心的先行指标

**这是消费股最重要的先行指标。**

```
合同负债 = 经销商预付的货款，货还没发
→ 合同负债增加 = 经销商抢着拿货，愿意提前打款
→ 合同负债减少 = 经销商观望，减少备货
```

**海天 2025**：
```
合同负债 41.3 亿（↓2.1 亿）
→ 同比减少约 5%
→ 经销商打款节奏略有放缓
→ 但这不算危险信号（变动在正常波动范围内）
→ 需要关注 Q1 数据：如果继续减少 > 10%，说明动销在变慢
```

### 2.4 销售费用率——增长是靠品牌还是一直砸钱

```
销售费用率 = 销售费用 / 营收

下降 → 品牌拉力增强（不需要花更多钱推广就能增长）→ 好信号
上升 → 要么在做品牌投入（好的上升），要么在打价格战（坏的上升）
```

**海天 2025**：海天的销售费用率长期在 5-7%，属于成熟品牌特征。对比新消费品牌动辄 20-30% 的销售费用率，成熟消费品不需要靠砸广告维持增长。

### 2.5 经销商数量和单产——渠道效率

```
经销商单产 = 营收 / 经销商数量
```

**海天 2025**：
```
经销商 6,702 家（净减少 5 家）→ 稳定
营收 289 亿 → 单经销商产出约 430 万/年

如果经销商数量增加但单产下降 → "稀释式扩张"——新经销商质量差
如果经销商数量稳定但单产上升 → "效率式增长"——渠道在变强

海天是后者，这是最好的增长模式
```

## 三、海天增长的三引擎拆解

### 引擎 1：量——还有多少空间？

```
酱油：149 亿（+8.55%）
→ 市场渗透率已超 95%，量的增长空间有限
→ 8.55% 的增速中，大约一半是真实量增（餐饮端恢复），一半是结构升级（高端酱油占比提升）
→ 未来 5 年量增大概率在 3-5%

蚝油：49 亿（+5.48%）
→ 蚝油渗透率 ~60-70%（低于酱油的 95%），还有空间
→ 但 5.48% 的增速不算高

调味酱：29 亿（+9.29%）
→ 竞争格局分散，老干妈是绝对龙头
→ 海天在调味酱上不是老大，但增速不错

其他品类：47 亿（+14.55%）⭐
→ 这是最大的亮点
→ 醋、料酒、复合调味料等"酱油以外"的品类
→ 增速是酱油的 2 倍
```

### 引擎 2：价——提价能力有多强？

```
海天历史上提价周期：约 2-3 年一次，每次 5-8%
2017 年：提价约 5%
2021 年：提价约 5-7%
2024-2025：没有再提价（原料成本下降，不需要提）

关键问题：2025 年的毛利率改善来自原料降价，那下次原料涨了，海天敢不敢提价？
→ 过去三次提价周期证明"敢"
→ 但提价的频率在变慢（市场对涨价更敏感了）
→ 价增不是海天未来的主驱动
```

### 引擎 3：品类扩张——这才是长期故事

```
海天不是酱油公司，是"调味品平台"
→ 酱油（52%）+ 蚝油（17%）+ 调味酱（10%）+ 其他（16%）
→ "其他"增速 14.55% >> 酱油 8.55%
→ 品类扩张的驱动力是：同一条渠道（6,700 家经销商、300 万个终端网点）可以卖越来越多的品类

这就是消费品最强大的商业模式——渠道复用。
渠道建好了，每加一个品类，边际成本极低。
```

## 四、消费股的投资时钟

```
渗透率 < 20% → 黄金期（量价齐升，PEG > 2 也合理）
渗透率 20-50% → 白银期（量增放缓，价增接力，PEG 1-1.5）
渗透率 > 50% → 成熟期（量价都慢，看品类扩张，PEG 0.8-1.2）
渗透率 > 80% → 成熟后期（几乎不增长，纯吃股息 + 分红率）
```

**海天当前**：
```
酱油渗透率 > 90% → 成熟后期
蚝油 ~60-70% → 成熟期
复合调味料 ~30-40% → 白银期

→ 海天整体处于"成熟期转慢增"阶段
→ 成长故事从"酱油放量"变成了"品类平台"
→ 25-30 倍 PE 的估值是合理的——对应慢增长 + 强护城河
→ 如果复合调味料做成了（10-15% CAGR 维持 5 年），估值还有上行空间
→ 如果品类扩张失败，估值会向纯酱油公司看齐（15-20 倍）
```

## 五、海天 2025 综合评估

用晨星护城河框架：

```
□ 护城河定性：品牌（无形资产）+ 成本优势（规模经济）
   → 品牌：海天酱油 = 酱油的代名词，提价不丢客户
   → 成本优势：产能 400 万吨+（对手最大的不到 100 万吨），规模优势不可复制
   → 渠道锁定：300 万个终端网点，新品牌根本铺不进去

□ 护城河宽度：宽阔
   → 酱油行业的格局（海天 > 25%，第二名 < 10%）几乎不可撼动
   → 规模优势 + 品牌认知 + 渠道密度 = 三重壁垒

□ 护城河趋势：稳定偏正面
   → 品类扩张在加速（其他 +14.55%）
   → 灯塔工厂降本增效（结构性的）
   → 渠道丝毫无损（6,700 家经销商稳定）

□ 假护城河排查：
   ✅ 品牌是真的（提价历史证明）
   ✅ 高份额是真的（不是降价换来的）
   ⚠️ ROE 下降是 H 股上市所致，不是经营恶化

□ 进化分类：再投资护城河（壁垒强 + 品类扩张让利润能以高 ROIC 再投入）
```

**投资逻辑一句话**：海天是 A 股最接近"永续消费品"的公司之一。买的是 10 年以上的品牌复利 + 品类扩张。短期催化剂是餐饮复苏 + 原料成本低位。

## 六、分析消费股的 15 分钟检查清单

```
⏱ 0-5 分钟：盈利质量
  → 毛利率趋势？（上升/下降/稳定？为什么？）
  → 净利率在行业中什么水平？
  → 经营现金流/净利润？（消费品通常 > 1.0）

⏱ 5-10 分钟：渠道先行信号
  → 合同负债增加还是减少？（经销商信心）
  → 经销商数量变化？（扩张还是收缩？单产升还是降？）
  → 应收账款/营收？（消费品应收通常 < 10%）

⏱ 10-15 分钟：增长质量
  → 营收增长来自量还是价？（量 > 价 > 涨价不可持续）
  → 品类扩张速度？（新品类增速是核心增长引擎）
  → ROE 杜邦拆解（消费品 ROE 应靠净利率+周转率，不是靠杠杆）
```

---

**Why:** 用户持有海天味业，但消费品的渠道和品类逻辑需要专门的框架来分析——不是看 PE 就够了，要看合同负债（先行指标）、经销商变化（渠道效率）、品类扩张（增长引擎）。

**How to apply:** 每季度跟踪海天季报时，核心看三个先行信号：合同负债 → 经销商信心，分品类增速 → 品类扩张进展，毛利率 → 竞争格局变化。
