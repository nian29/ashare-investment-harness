# 电力行业分析框架

> 配合 [[investment-handbook]] 维度一和维度二使用。
> 电力是 A 股最特殊的行业之一——利润不是"卖得多就赚得多"，产品（电力）同质化、价格大概率被管制、增长靠的是装机扩张而非同店增长。

---

## 一、电力的生意本质

电力公司赚的不是"卖电"的钱——因为电是同质化的，谁也卖不出溢价。**赚的是"发电资产"的钱：**

```
利润 = 发电量 × 上网电价 − 成本
     = (装机容量 × 利用小时数) × 上网电价 − (折旧 + 燃料 + 运维 + 财务费用)
```

**利润公式里四个变量的行业特性**：

| 变量 | 特性 | 谁决定 |
|------|------|--------|
| 装机容量 | 建好就不能变 | 资本开支决策（几年前就定了） |
| 利用小时数 | 每年波动 | 电网调度 + 来水/来风/日照 + 检修 |
| 上网电价 | 非市场决定 | 政策 + 市场化交易比例 |
| 成本 | 每种电源不同 | 煤价/折旧/运维 |

**核心结论**：看电力公司，不是看"今年卖了多少电"，而是看"有多少机组在什么阶段（在建/调试/商运）"。因为从决定建一个电厂到它开始赚钱，需要 2-8 年。

## 二、四种电源，四种生意

你的电力 ETF 持仓涵盖了四种电源。每一种的盈利逻辑完全不同。

### 2.1 🔴 火电：赌煤价

```
利润 = 发电量 × 上网电价 − 煤炭成本 − 折旧 − 运维

最大变量：煤价（燃料成本占火电总成本的 60-70%）
次要变量：利用小时数（火电是调峰主力，风光多了火电就少发）
```

**投资逻辑**：
- 煤价跌 → 火电利润暴增（反之，煤价涨 → 利润被吃掉）
- 所以火电是**周期股**，不是公用事业股
- 火电 PE 最低的时候，往往是煤价最高的时候 → 周期股的经典陷阱

**当前（2026.6）**：油价/煤价下行 → 利好火电成本端。

### 2.2 🔵 水电：印钞机

```
利润 = 发电量 × 上网电价 − 折旧 − 运维

最大变量：来水量（天给的，无法控制）
成本特性：折旧占总成本 40-50%，但折旧不是现金支出
          → 经营现金流 >> 净利润
```

**投资逻辑**：
- 水电是一次性的巨额投入（大坝），之后几十年的变动成本几乎为零
- 把折旧期熬过去后（一般 30-40 年），利润会跳升（折旧提完，成本骤降）
- 长江电力就是典型——高股息 + 稳定现金流 + 不可复制的地理位置
- **水电的护城河**是地理垄断——好的坝址全世界就那几个

### 2.3 🟢 核电：赌政策 + 赌装机增长

```
利润 = 发电量 × 上网电价 − 折旧 − 核燃料 − 运维 − 财务费用

最大变量：
  - 上网电价（当前 71% 进入市场化交易，电价在下行）
  - 折旧（核电折旧年限 25-40 年，折旧提完后 ROE 会跳变）
  - 财务费用（一台机组 200 亿+，利息支出巨大）
```

**中国核电 2025 实战拆解**：

```
装机：核电在运 25GW + 在建/核准 18 台（意味着装机还能翻倍）
利用小时数：> 8,000 小时（历史最优，核电是基荷电源，几乎不停）
上网电价：0.3899 元/度（↓6.27%）

收入 = 1,878 亿度 × 0.3899 元 ≈ 732 亿
实际核电收入 671 亿 → 说明有一些电没上网或电价更低

归母净利润 93 亿（+6%）
→ 但 ROE 只有 8.21% → 为什么这么低？
→ 因为大量在建机组还没产生收益，但已经背了巨额债务
→ 一旦在建机组投运（18 台），利润会阶梯式跃升
→ 但财务费用也会同步增加（需要 3,000 亿+ 投资）

Q1 净利润暴跌 34% 的原因：
→ 大修机组 7 台（同比增加 5 台）→ 发电量减少
→ 电价持续下行
→ 但这是短期扰动，大修完就恢复
```

**核电的投资逻辑是一种"痛苦的等待"**：
- 你知道 18 台在建机组迟早会投运 → 利润会跳升
- 但你不知道机制电价（核电的特殊电价政策）什么时候落地
- 在等待期间，市场化电价下行会不断侵蚀利润
- **这是典型的"时间是你的朋友，但过程很折磨"的投资**

**核电 ROE 的终局推演**：
```
当前：8.21%（大量在建机组拖累）

如果 18 台在建全部投运：
→ 装机翻倍 → 收入翻倍
→ 但折旧+财务费用也翻倍 → 利润不一定翻倍
→ ROE 可能在 10-12% 区间（仍然不高，因为核电是重资产）

更关键的变量不是收入，而是电价：
→ 机制电价如果落地（保障一定利润率），ROE 会大幅跃升
→ 如果纯市场化，ROE 可能长期在 8-10%
```

### 2.4 🟡 风电/光伏：赌利用小时数 + 补贴

```
利润 = 发电量 × 上网电价 − 折旧 − 运维

最大变量：利用小时数（风来不来、太阳出不出，每年都不一样）
次要变量：补贴回款（很多新能源项目的应收账款是拖欠的补贴）
```

**投资逻辑**：
- 风电/光伏是成长最快的电源（装机增速 > 20%），但也是最不稳定的
- 适合景气投资：装机增长快的时候买，增速放缓就卖
- 不太适合长期持有吃股息（来风/日照的波动太大）

## 三、电价机制——这个行业最大的政策变量

电力行业不是"自由市场"，电价是政策+市场双重决定的。

### 三种电价

| 电价类型 | 怎么定 | 谁说了算 |
|---------|--------|---------|
| **计划电价** | 政府核定，固定 | 发改委 |
| **市场化交易电价** | 发电企业和用户谈 | 市场供需（但天花板被政策限定了） |
| **机制电价**（核电在推）| 保障一定回报率的电价 | 政策试点中 |

**中国核电 2025 的关键变化**：
```
市场化交易比例：71.36%（同比+20.58pct）→ 大幅提升
→ 这意味着核电从"固定电价"向"浮动电价"转变
→ 在电力供过于求的环境下，市场化 = 降价
→ 核电电价 -6.27% 就是这个原因

机制电价的逻辑：
→ 核电建一台机组要 200-300 亿，需要 8 年
→ 没有稳定的电价预期，没人敢投资
→ 所以政策需要给核电一个"保障回报率"的电价
→ 正在推的机制电价就是想解决这个问题
→ 如果落地，核电的盈利稳定性会大幅提升，估值也应该上调
```

### 煤电联动 vs 核电机制电价

```
火电：煤价涨 → 政策允许电价涨（煤电联动），但滞后且不充分
核电：现在推机制电价 → 保障 8-9% ROE 的电价水平
水电：计划电价为主，市场化比例低，电价波动小
```

## 四、电力 ETF（561560）应该怎么看

你的电力 ETF 不是单押任何一种电源，是一个电力行业的"宽基指数"。

**估算构成**：
```
火电    ~30%：赌煤价下行 + 调峰价值重估
水电    ~20%：稳定现金流 + 高股息（长江电力 9%）
核电    ~13%：装机增长 + 政策催化（中国核电 + 广核）
风光    ~15%：高成长 + 高波动
电网/其他 ~12%：储能 + 电网设备
```

**这意味着**：
- ETF 的短期波动主要来自火电（煤价）和风光（利用小时数）
- ETF 的长期回报来自水电（分红）+ 核电（装机增长）
- **这不是一只"买了就不管"的 ETF**——你需要关注煤价走势和电价政策

## 五、分析电力公司的 15 分钟检查清单

```
⏱ 0-5 分钟：装机与电量
  → 在运装机容量？在建/核准规模？（未来 3-5 年能增长多少）
  → 利用小时数？（> 设计值 = 运营优秀）
  → 发电量增速？（跟自己比，不用跟别人比）

⏱ 5-10 分钟：电价与成本
  → 上网电价趋势？（升/降/稳？为什么？）
  → 市场化交易比例？（越高越受市场波动影响）
  → 燃料成本/折旧/财务费用占比？（哪种电源看哪个变量）

⏱ 10-15 分钟：分红与回报
  → 分红比例？股息率？
  → ROE 趋势？（上升 = 在建转商运，下降 = 新项目拖累）
  → 经营现金流/净利润？（> 2.0 正常，因为折旧不是现金支出）
```

---

**Why:** 用户持有电力 ETF 且关注核电买点，但缺少电力行业的特化分析框架。电力行业的特殊性（价格管制、重资产、长周期）决定了一套独立的分析逻辑。

**How to apply:** 分析任何电力标的时，先识别属于哪种电源类型（不同逻辑完全不同），然后拆解利润公式（装机×小时×电价−成本）。
