# 高端制造/军工行业分析框架

> 高端制造是 A 股第三大板块（仅次于消费和金融），军工是高端制造里最特殊的子集。这两个行业的共同特征：**政策是大腿，订单是氧气，技术自主可控是护城河。**

---

## 一、高端制造 vs 传统制造——区别在哪

| | 传统制造 | 高端制造 |
|------|---------|---------|
| **产品** | 标准化、大宗化 | 定制化、高技术壁垒 |
| **护城河** | 成本优势（规模） | 技术壁垒 + 客户认证 |
| **毛利率** | 15-25% | 35-60% |
| **增长驱动** | 价格 + 量（产能） | 技术迭代 + 下游 CapEx |
| **客户** | 分散 | 高度集中（大客户依赖） |
| **估值** | PE 10-20x | PE 30-60x（含技术溢价） |
| **A 股代表** | 钢铁、水泥、化工 | 半导体设备、军工、工业自动化 |

## 二、高端制造的通用分析框架

### 2.1 四个核心问题

**问题 1：技术是"能做的"还是"只有我能做的"？**

```
能做的（竞争对手也能做）→ 打价格战 → 毛利率会趋势性下降 → 不配高估值
只有我能做的 → 技术壁垒 → 毛利率稳定甚至上升 → 配高估值

A 股实操：
→ 半导体设备（如中微公司刻蚀机）：全球只有 2-3 家能做 → 高壁垒
→ 普通 CNC 机床：几百家能做 → 低壁垒
```

**问题 2：客户是谁？认证壁垒有多高？**

```
大客户（如台积电、苹果、中芯国际）的供应商认证需要 3-5 年
→ 一旦进去，客户不会轻易换（转换成本极高）
→ 新进入者拿不到认证 → 已有的玩家享受"认证护城河"
→ 这比专利护城河更牢固（专利会过期，认证关系不会）

但大客户依赖也是风险：
→ 如果唯一客户是台积电 → 台积电砍单 = 公司收入腰斩
```

**问题 3：订单可见度有多长？**

```
高端制造的订单通常提前 6-18 个月下单
→ 在手订单/营收 > 1.5x = 未来 1-2 年收入基本锁定 → 确定性高
→ 在手订单/营收 < 0.8x = 需要不断拿新订单 → 不确定性高

这是高端制造最重要的先行指标。
```

**问题 4：国产替代空间多大？**

```
当前国产化率低 + 技术正在突破 → 国产替代故事成立 → 估值弹性大
当前国产化率已经很高 → 竞争格局趋于稳定 → 增长靠行业整体扩张

A 股例子：
→ 半导体设备国产化率 ~20% → 空间大 → 未来 5 年 CAGR 可能 30%+
→ 光伏设备国产化率 ~95% → 空间小 → 增长只能靠海外扩张
```

### 2.2 高端制造的估值

```
PE 不是最重要的——因为高端制造利润波动大（研发费用、折旧、新项目投产）

更重要的：
→ PS（市销率）：制造业的收入比利润稳定
→ EV/EBITDA：排除折旧政策差异（高端制造折旧是大头）
→ 在手订单/市值：这个比率越高，公司越"便宜"

高端制造的 PEG：
→ PEG < 0.8 = 可能值得关注（增速 30%+ 是正常的）
→ PEG > 1.5 = 偏贵（除非技术壁垒极深）
```

## 三、军工——高端制造里最特殊的子集

### 3.1 军工和其他行业的本质区别

| | 普通制造业 | 军工 |
|------|----------|------|
| **客户** | 市场化（企业） | 单一买家（军方/政府） |
| **定价** | 市场价格 | 成本加成（政府定价） |
| **利润天花板** | 市场决定 | 政策决定（利润率通常被锁定在 5-10%） |
| **订单周期** | 6-18 个月 | 3-5 年（长周期） |
| **技术迭代** | 市场化驱动 | 国家驱动 |
| **出口** | 自由出口 | 严格管制 |
| **估值方法** | PE/PS/PEG | 订单倍数 + PB（不能用 PE） |

**核心结论**：军工股的利润不是市场的产物，是政策的产物。所以不能用市场化逻辑去分析。

### 3.2 军工股的两个核心指标

**指标 1：订单倍数**

```
在手订单 / 年营收 = 订单倍数

订单倍数 > 3x → 未来 3 年收入高度确定 → 低风险
订单倍数 1.5-3x → 正常
订单倍数 < 1x → 需要依赖新订单，不确定性高
```

**指标 2：合同负债（预收款）**

```
军工的合同负债跟消费品的含义不同：
→ 军方的预付款通常占合同总额的 30-50%
→ 合同负债暴增 = 大额订单到手的信号
→ 这份合同会持续 3-5 年，提供长期收入确定性

跟踪方法：每季度看资产负债表，合同负债跳升 → 收到新订单了
```

### 3.3 军工的估值

```
不能用 PE：
→ 军工的利润率可能被政策锁定在 5%
→ 但 PE = 利润 × 倍数 → 利润被压低 → PE 看起来很高 → 实际上不一定贵

更合理的：
→ PB（市净率）：军工是重资产行业，资产质量相对透明
→ 订单倍数估值：市值 / 在手订单
→ EV / 未来 3 年合计利润
```

### 3.4 军工的关键催化剂和风险

```
✅ 催化剂：
  → 军费预算增长（每年 3 月两会公布）
  → 新型号装备列装（如新战机、新舰艇批量生产）
  → 出口突破（获得海外大单）
  → 军民融合政策

❌ 风险：
  → 军费增速放缓
  → 定价机制改革（成本加成 → 可能会进一步压缩利润）
  → 客户单一（军方）→ 付款节奏不确定 → 应收账款高
  → 地缘政治出口管制 → 技术封锁 → 上游卡脖子
```

## 四、高端制造/军工检查清单

```
⏱ 0-5 分钟：行业定位
  → 高端制造还是普通制造？（毛利率 > 35% 是分水岭）
  → 属于哪个赛道？（半导体设备/军工/工业自动化/新能源设备？）
  → 技术壁垒：全球有几家能做？

⏱ 5-10 分钟：订单和客户
  → 在手订单/营收？（> 1.5x = 确定性强）
  → 前五大客户占比？（> 70% = 客户集中度风险）
  → 最大客户是谁？认证壁垒有多高？

⏱ 10-15 分钟：估值和空间
  → 订单倍数（军工专用）
  → 国产替代空间（低国产化率 = 高成长空间）
  → 当前估值在历史什么位置？（PS/EV 更好）
```

---

**Why:** 高端制造和军工是 A 股第三大板块，用户目前完全没有覆盖。这两个行业的逻辑（订单驱动+客户认证壁垒+政策定价）跟消费品和银行完全不同。

**How to apply:** 分析高端制造公司时，核心看技术壁垒（全球几家能做？）+ 订单可见度（在手订单/营收）+ 客户认证壁垒。军工额外看订单倍数和合同负债跳升信号。
