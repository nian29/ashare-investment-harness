# 科技/半导体行业分析框架

> 配合 CPO/光模块讨论使用。科技股不是不能用框架，是框架跟消费/银行完全不同——利润不是核心变量，技术路线和产能周期才是。

---

## 一、科技的生意本质——三个周期叠加

科技/半导体是三个周期的叠加，每个周期长度不同：

```
短周期（库存周期）  ：3-5 个季度 → 决定季度利润波动
中周期（资本开支周期）：3-5 年     → 决定产能过剩/不足
长周期（技术周期）   ：10-20 年   → 决定谁被淘汰、谁崛起
```

**这就是为什么科技股波动巨大**——你同时在赌三个时间维度的方向。

---

## 二、短周期——库存周期（最直接影响季度业绩）

半导体/光模块的库存周期是所有行业里最剧烈的：

```
补库存（Up Cycle）：
→ 下游需求好 → 客户加单 → 芯片厂加班 → 供不应求 → 涨价 → 利润暴增
→ 持续时间：通常 4-6 个季度

去库存（Down Cycle）：
→ 客户库存太多 → 砍单 → 芯片厂产能空转 → 降价清库存 → 利润暴跌
→ 持续时间：通常 2-4 个季度
```

**当前（2026.6）光模块行业库存状态**：
```
光模块：仍处于补库存阶段（AI 算力需求驱动）
→ 中际旭创 Q1 营收 +192%，说明下游还在抢货
→ Lumentum 说产能要卖到 2028 年 → 补库存远未结束
→ 最大的风险：什么时候客户说"库存够了"
  → 目前没有任何信号
  → 但当这个信号出现时，Q1 192% 的增速会变成负增长
```

### 库存周期的先行指标

```
领先信号（1-3 个月前预警）：
→ 客户资本开支指引下调（微软/Meta/Google 的 CapEx 指引）
→ 渠道库存天数开始上升
→ 芯片交货周期（Lead Time）开始缩短（供需反转的信号）
→ 现货价格开始松动

滞后信号（出来后已经跌了 20%）：
→ 季报利润暴跌
→ 公司正式宣布裁员/减产
```

## 三、中周期——资本开支周期（决定 2-3 年后的供需）

这是 Serenity 框架的核心——追踪大厂 CapEx。

```
当前 AI CapEx 周期：
→ 2025 年全球六大 CSP CapEx ~2600 亿美元
→ 2026 年预计 ~6000+ 亿美元（+78%）
→ 光模块市场 2026-2028：332→525→640 亿（华泰预测）

关键问题：
→ CapEx 增速 78% 能持续吗？不能。
→ 什么时候增速放缓？不知道。但放缓的那天，就是光模块周期见顶的时候。
```

### CapEx 周期的判断框架

```
加速期（增速从 20% → 100%）：光模块需求暴增 → 供不应求 → 业绩爆发
  → 当前阶段 ← 你在 CPO 讨论里所处的

减速期（增速从 100% → 20%）：需求仍在增长，但加速度下降
  → 估值开始收缩（PE 从 114x → 60x → 30x）
  → 股价可能不跌，但因为增速放缓，PE 被重新定价

收缩期（增速变负）：产能过剩 → 砍单 → 价格战 → 利润暴跌
  → 光模块的"冬天"

恢复期（增速由负转正）：产能出清 → 供需修复 → 下一轮周期开始
```

## 四、长周期——技术路线（决定 5-10 年后谁活着）

这是科技股最可怕的变量。

```
光模块技术路线：
800G → 1.6T → 3.2T —— 每一代产品升级，竞争格局都可能洗牌

CPO vs 可插拔：
→ 可插拔是当前主流（中际旭创的核心业务）
→ CPO 是下一代方向（天孚通信的核心卡位）
→ 如果 CPO 加速取代可插拔 → 中际旭创的大部分产品线过时 → 估值需要重写

硅光 vs VCSEL：
→ 两种不同的激光器技术路线
→ 中际旭创和天孚用的是硅光路线
→ 如果 VCSEL 路线胜出 → 硅光企业受损

技术路线的判断标准：
→ 不是"哪个技术更好"，是"哪个技术被英伟达/博通选中"
→ 英伟达 Rubin 平台选了硅光+CPO → 硅光路线目前领先
```

## 五、科技股的核心分析指标（和消费/银行完全不同）

### 不用看的

```
❌ PE：利润太不稳定，PE 没有意义（中际旭创 PE 从 30x 涨到 114x 不是因为"变贵了"，是因为利润还没出来）
❌ PB：轻资产，PB 没意义
❌ 股息率：大多数科技股不分红（钱要投回研发）
```

### 必须看的

```
✅ 营收增速：比利润增速更重要。利润可以通过压缩费用做出来，收入不会骗人。
  → 中际旭创 Q1 营收 +192% → 需求是真的

✅ 毛利率趋势：科技股的"护城河"指标。
  → 毛利率上升 = 产品有竞争力，客户愿意付溢价
  → 毛利率下降 = 产品在大宗化，竞争加剧
  → 中际旭创 2025 毛利率 42.6%（↑）→ 产品升级在推高毛利率

✅ 研发费用/营收：科技股的"未来"指标。
  → 把研发费用砍了，当期利润好看，但 2 年后没有新产品 → 死路
  → 研发费用率持续下降 ← 危险信号（在吃老本）

✅ 前五大客户集中度：科技股最大的隐藏风险。
  → 中际旭创 > 50% 收入来自英伟达 → 单一客户风险极高
  → 天孚通信也是高客户集中度

✅ 在手订单/合同负债：科技股的"先行指标"。
  → 比消费股的合同负债更重要
  → 光模块的交货周期长，订单提前 6-12 个月就能看到
```

## 六、CPO/光模块行业检查清单

```
⏱ 0-5 分钟：技术路线
  → 公司的产品属于哪个技术路线？（硅光 vs VCSEL）
  → 谁是最大客户？（英伟达占比多少？）
  → 技术路线变化对这家公司是利好还是利空？

⏱ 5-10 分钟：周期位置
  → 营收增速是在加速还是减速？
  → 客户 CapEx 指引是在上调还是下调？
  → 库存周期在什么阶段？

⏱ 10-15 分钟：估值
  → 用 PS（不好用 PE） + 反向 DCF
  → 当前价格在赌 CapEx 继续高增长的假设
  → 如果 CapEx 增速从 78% 降到 20%，这只股票值多少钱？
```

---

**Why:** 用户讨论过 CPO 但没有科技/半导体行业的特化框架——用消费品的 PE 逻辑看科技股会完全失效。本手册提供三个周期叠加的分析框架和科技股特化指标。

**How to apply:** 分析任何科技股时，先判断三个周期的位置（库存+CapEx+技术路线），再看特化指标（营收增速>毛利率>在手订单>客户集中度）。PE 不重要。
