# 周期股投资框架

> 配合 [[industry-power-framework]]（火电部分）和 [[industry-banking-framework]] 使用。
> 周期股的核心悖论：PE 最低的时候最危险，PE 最高的时候反而是买点。

---

## 一、周期股的生意本质

周期股和非周期股的根本区别：

```
非周期股（海天）：需求稳定，利润波动来自公司自身经营
周期股（火电）：需求/成本/价格随宏观大幅波动，利润更多取决于"你在周期的哪个位置"而不是"公司经营多优秀"
```

**周期股的利润公式里，有一个公司完全控制不了的变量**：

| 周期行业 | 不可控变量 | 变量波动幅度 |
|---------|----------|------------|
| 火电 | 煤价 | ±50% |
| 钢铁 | 钢价 + 铁矿石价格 | ±40% |
| 化工 | 产品价格 + 原料价格 | ±50% |
| 养猪 | 猪价 | ±100% |
| 航运 | 运价 | ±200% |
| 银行 | 利率 + 信用成本 | ±30% |

## 二、周期阶段的四象限

```
                    景气度 ↑
                       │
      复苏期           │          繁荣期
   （利润刚开始涨）     │      （利润持续新高）
   PE 看起来很高        │      PE 看起来很低
                       │
  ──────────────────────────────────→ 时间
                       │
      衰退期           │          低谷期
   （利润开始跌）       │      （利润极低甚至亏损）
   PE 开始升高          │      PE 极高或为负
                       │
```

### 四阶段的操作逻辑

| 阶段 | 信号 | PE 表现 | 操作 |
|------|------|---------|------|
| **低谷** | 全行业亏损、产能出清、没人讨论 | PE 极高或负 | 🟢 最佳买点（逆向） |
| **复苏** | 产品价格开始涨、产能利用率回升 | PE 从高向下回落（利润涨得更快） | 🟢 买（趋势确立） |
| **繁荣** | 全行业暴赚、扩产计划频出、媒体热捧 | PE 极低（利润在历史最高） | 🔴 卖（PE 低 = 陷阱） |
| **衰退** | 产品价格松动、库存积压 | PE 从低向上爬（利润跌得更快） | 🔴 卖或做空 |

**核心认知**：周期股的 PE 跟价值股是反的——PE 低的时候是卖点（利润在巅峰），PE 高的时候是买点（利润在谷底）。

## 三、判断周期位置的五个先行指标

### 1. 产品价格趋势

最直接、最领先。火电看煤价，钢铁看钢价，养猪看猪价。

```
当前（2026.6）：煤价/油价下行 → 火电成本端改善 → 火电利润在复苏期
但煤价不可能永远跌 → 要提前想：煤价什么时候见底？
```

### 2. 产能利用率/行业开工率

价格涨 → 开工率升 → 新产能开始规划 → 2-3 年后产能集中释放 → 价格暴跌。

```
规律：行业开工率 > 85% + 大量新项目宣布 → 离顶部不远了
     行业开工率 < 60% + 大量产能关停 → 离底部不远了
```

### 3. 行业资本开支

**最重要的中期领先指标**——从决策投资到产能释放通常需要 2-3 年。

```
行业 CapEx 增速 > 30% → 2-3 年后产能过剩 → 利润暴跌
行业 CapEx 增速 < 0%（负增长）→ 2-3 年后产能不足 → 利润暴涨

当前火电：煤价下行 → 火电利润改善 → CapEx 可能增加 → 警惕 2-3 年后
```

### 4. 库存

```
主动去库存（库存降 + 价格跌）→ 衰退期，最痛苦
被动去库存（库存降 + 价格稳/涨）→ 复苏早期，好买点 ← 当前煤价？
主动补库存（库存升 + 价格涨）→ 繁荣期，警惕
被动补库存（库存升 + 价格跌）→ 衰退起点，快跑
```

### 5. 分析师一致预期

```
行业亏损时：分析师一片悲观 → 可能接近底部
行业暴赚时：分析师一致上调 → 可能接近顶部
```

## 四、你的持仓中的周期属性

### 电力 ETF 中的火电部分（~30%）

```
火电 = 纯周期股（赌煤价方向）

当前（2026.6）：
→ 煤价/油价下行 → 火电成本改善
→ 火电处于 "复苏期"（利润在从底部恢复）
→ PE 可能看起来偏高（因为利润还没完全释放）

操作含义：
→ 如果你相信煤价继续跌 → 火电还在复苏早期，可以持有
→ 如果煤价见底回升 → 火电利润见顶，应该减仓
→ 火电不是"买了就不管"的持仓，煤价是必须跟踪的变量
```

### 电力 ETF 中的水电/核电部分（~33%）

```
水电/核电 = 弱周期（防御型公用事业）
→ 利润波动小，不适用周期框架
→ 按公用事业方法持有（看装机增长 + 股息率）
```

### 农业银行

```
银行 = 弱周期（受信用周期影响）

银行的周期不来自价格，来自信用：
→ 经济扩张期：贷款需求旺 + 坏账少 → 利润好
→ 经济收缩期：贷款需求弱 + 坏账多 → 利润差

当前（2026.6）：
→ 净息差在历史低位（1.28%）= 接近周期底部
→ 不良率在改善（1.27%↓）= 信用周期在好转
→ 银行处于 "复苏早期"

操作含义：
→ 息差见底 + 不良改善 = 银行盈利周期在底部向上
→ 现在持有银行是对的（周期有利 + 估值低）
→ 但息差修复会很慢——这是缓慢的复苏，不是 V 型反转
```

## 五、周期股投资铁律

1. **不要在 PE 最低的时候买。** PE 极低 = 利润在历史峰值 = 离顶部不远。

2. **在没人讨论、全行业亏损的时候买。** 逆向——马克斯的核心教训。

3. **盯住 CapEx 周期。** 当所有人都在扩产 → 准备卖。当所有人都在关产能 → 准备买。

4. **周期股不适合永远持有。** 周期到了顶部必须卖，不能"长期持有等分红"。

5. **周期底部是过程不是点。** 底部可能持续 1-2 年，不要试图精准抄底——在底部区间慢慢买。

6. **区分"周期问题"和"结构问题"。** 煤价跌了还会涨 = 周期问题（可以等）；火电被光伏+储能永久替代 = 结构问题（不能等）。
