# 彼得·林奇 + 邱国鹭框架展开

> 手册里提了林奇（买你懂的、生活选股、tenbagger）和邱国鹭（数月亮不数星星、便宜是硬道理），但没有展开。这篇把两个框架具体化，嵌入 A 股实践。

---

## 第一部分：彼得·林奇框架

林奇在富达基金 13 年年化 29%，《彼得·林奇的成功投资》是三本经典中最实操的一本。

### 1.1 六种股票分类——这是林奇框架最核心的贡献

**同一只股票，不同阶段属于不同类型。分类错了，操作就全错了。**

| 类型 | 特征 | 预期回报 | 看什么 | 什么时候卖 |
|------|------|---------|--------|----------|
| **缓慢增长型** | 增速 2-4%，成熟行业 | 股息率 + 2-3% | 分红是否稳定 | 基本面恶化或找到更好的 |
| **稳健增长型** | 增速 10-12%，龙头公司 | 年均 10-15% | PE 是否合理 | PE 远超历史中枢 |
| **快速增长型** | 增速 20%+，成长期 | 年均 20%+ | 增长还能持续多久 | 增速放缓（不是等它停了再卖） |
| **周期型** | 利润随经济周期波动 | 低估到高估的差价 | 周期所处位置 | 周期见顶信号出现 |
| **困境反转型** | 从亏损/困境中恢复 | 翻倍级 | 反转确定性 | 反转兑现 |
| **隐蔽资产型** | 有市场没注意到的资产 | 资产重估 | 资产真实价值 | 资产被发现+重估完成 |

### 1.2 你的持仓分类练习

```
农业银行 → 缓慢增长型
  → 增速 3%、关注分红、PB < 1 才买、当股息率不再有吸引力时卖
  → 核心投资回报来自股息 + PB 从 0.8 修复到 1.0 的估值收益

海天味业 → 缓慢增长型 转 稳健增长型
  → 酱油增速 8.5% = 缓慢增长
  → 其他品类增速 14.55% = 稳健增长
  → 整体在缓慢增长和稳健增长的边界上
  → 看 PE 中枢（25-30 倍），超过就卖

中国核电 → 稳健增长型
  → 装机增速支撑，但电价在侵蚀利润
  → 增长确定性高但速度不快（6%+）

CPO（中际旭创） → 快速增长型
  → 增速 100%+，但这个增速不可能持续
  → 危险：快速增长型一旦增速放缓，PE 会从 100 倍杀到 30 倍
  → 林奇提醒："对快速增长型，不是要卖在最高点，是要在增速开始放缓时就卖"
```

### 1.3 PEG 系统的正确用法

林奇最著名的公式：**PEG = PE / 增长率**。PEG < 1 是便宜。

**但 PEG 有三个陷阱，林奇专门警告过**：

**陷阱 1：增长率用的是"未来"不是"过去"**
```
错误：海天 PE 27 / 去年增速 11% = PEG 2.45 → 太贵了
正确：海天 PE 27 / 未来 3-5 年预期增速 ~12% = PEG 2.25 → 仍然偏贵
但：海天是成熟消费品，不能用快速增长型的 PEG 标准
    → PEG 最适合快速增长型（增速 > 20%），缓慢增长型不适用 PEG
    → 农行增速 3%，PEG = 8/3 = 2.67，没有意义
```

**陷阱 2：没有区分 PE 类型**
```
PE(TTM) = 基于过去 12 个月的利润（滞后）
PE(动态) = 基于当前财年的预期利润（前瞻）
PE(静态) = 基于上一财年的利润

PEG 要用动态 PE（前瞻），不能用 TTM PE
```

**陷阱 3：不同行业不能用同样的 PEG 标准**
```
消费龙头 PEG 1.5 = 合理（因为有护城河溢价）
一般制造 PEG 1.0 = 合理
周期股：PEG 在利润顶点特别低（因为 E 被利润高峰夸大了）→ 陷阱
```

### 1.4 林奇最实用的一条建议

> "投资你了解的东西。如果你不能在两分钟内跟一个 10 岁小孩解释清楚这家公司怎么赚钱，你就不该买。"

**对照你的持仓**：
- 农行："靠吸收存款、发放贷款赚利息差价" ✅ 两分钟能讲清
- 海天："靠卖酱油蚝油调味酱赚钱" ✅ 两分钟能讲清
- 核电："靠核反应堆发电卖电赚钱" ✅ 两分钟能讲清
- CPO 光模块："把激光器和芯片封装在一起，让 AI 服务器里的数据传得更快" → 勉强能讲清，但你真懂硅光 vs VCSEL 的技术路线吗？

## 第二部分：邱国鹭框架

邱国鹭是高毅资产创始人，《投资中最简单的事》作者。A 股本土实战派，方法论极其朴实。

### 2.1 "数月亮不数星星"

**一句话**：在胜负已分的行业买龙头，不要在胜负未分的行业赌黑马。

```
"数月亮"：行业格局清晰，第一名和第二名的差距一眼看得出来
"数星星"：行业格局混乱，每家都差不多，谁也不服谁

A 股实战：
✅ 白酒：茅台是月亮，后面全是星星 → 买茅台
✅ 酱油：海天是月亮 → 买海天
✅ 银行：四大行是月亮 → 买农行建行
❌ AI 应用：谁是月亮？不知道 → 不赌
❌ 预制菜：谁都可能做起来 → 不赌
```

**注意**：邱国鹭说这话的时候，很多行业已经涨了很多了。他的意思是"在月亮还不太贵的时候买"，不是"不管多贵都买月亮"。

### 2.2 "便宜是硬道理"

**一句话**：估值是 A 股最重要的因子。高估值是最大的敌人。

```
邱国鹭的原话逻辑：
→ 好公司 + 好价格 = 好投资（满分）
→ 好公司 + 坏价格 = 坏投资（关键错误）
→ 烂公司 + 好价格 = 可能的投资（但难度大，不推荐）
→ 烂公司 + 坏价格 = 灾难

对照你的持仓：
→ 农行 PE 8x = 好价格 → 农行不是"最优秀"的银行，但价格够好
→ 海天 PE 27x = 合理偏贵 → 海天是"好公司"，但价格不算"好价格"
→ CPO PE 114x = 坏价格 → 再好公司的好故事，这个价格都难有好结果
```

### 2.3 四个投资维度

邱国鹭把投资简化为四个问题：

| 维度 | 问题 | 你的农行答案 | 你的海天答案 |
|------|------|------------|------------|
| **估值** | PE/PB 贵吗？ | PB 0.81，便宜 | PE 27x，合理 |
| **品质** | 是不是好生意？ | 牌照垄断+转换成本，好 | 品牌+规模+渠道三壁垒，好 |
| **时机** | 现在对吗？ | 息差企稳中，时机尚可 | 原料成本低位+餐饮复苏，时机尚可 |
| **催化剂** | 什么能让它涨？ | 坏账下降、分红提升 | 品类扩张加速、消费刺激 |

**四个问题里，品质是慢变量（长期不变），估值和时机是快变量（每个季度都可能变）。**

### 2.4 邱国鹭的卖出原则

```
1. 基本面恶化（不管赚了多少都卖）
2. 估值太高（达到历史高位区就卖）
3. 找到更好的（有更便宜的好东西，换仓）
```

这跟林奇的卖出逻辑一致——**卖出的理由和买入价无关。**

## 三、林奇+邱国鹭的融合使用

两位老师的分工：

| 问题 | 用谁 |
|------|------|
| 这公司做什么的？我懂吗？ | 林奇（买你懂的） |
| 它属于哪种类型？该用什么策略？ | 林奇（六种分类） |
| 这行业胜负已分了吗？ | 邱国鹭（数月亮） |
| 价格对吗？ | 邱国鹭（便宜是硬道理）+ 马克斯（第二层思维） |
| 什么时候卖？ | 林奇（类型决定卖法）+ 邱国鹭（基本面/估值/替代） |

**融合公式**：
> 用邱国鹭选"月亮"（好行业的好公司），用林奇判断"类型"（该用哪种策略），用马克斯决定"价格"（什么时候买），用你自己的手册执行纪律。

## 四、林奇+邱国鹭的金句速记

### 林奇
- "投资你了解的东西"
- "不要拔掉鲜花浇灌杂草"（不要卖掉盈利的持仓去补仓亏损的）
- "PEG 是对快速增长型股票最有用的指标"
- "股市下跌就像暴风雪，是让你买到便宜货的机会"

### 邱国鹭
- "在胜负已分的行业买龙头，在胜负未分的行业买指数"
- "便宜是硬道理。好公司买贵了也是烂投资"
- "投资中最简单的事：好行业、好公司、好价格"
- "宁可错过，不要做错"
