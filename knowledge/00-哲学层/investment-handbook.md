## 一、投资哲学（底层操作系统）

### 三句话定调

1. **买得好比买好的更重要** — 定价决定一切，好公司买贵了也是烂投资
2. **在别人恐惧时贪婪，在别人贪婪时恐惧** — 逆向才是超额收益的来源
3. **防御优先，先活下来再谈赚钱** — 有老投资者，有胆大投资者，但没有老而胆大的投资者
4. **永远不要说"这次不一样"** — 八百年数据证明，金融危机的本质从不改变 [[financial-crisis-classics]]

### 四位老师的框架融合

| Serenity（白毛） | 瓶颈理论、CapEx溯源、碎片拼图、AI辩论 | 决定**买什么方向** |

## 二、选股分析框架（五维打分卡）

每只新标的按五个维度打分（每项1-5分），**总分≥20分 + 无危险信号 = 可进入关注池**。

### 维度一：商业模式（行业+护城河）

> 🔑 深度框架见 [[morningstar-moat-framework]] —— 五大护城河来源 + 趋势评级 + 假护城河识别

| 问题 | 好 | 差 |
|------|-----|-----|
| 行业赛道 | 成长型/刚需型 | 衰退型/过度竞争 |
| 竞争壁垒 | 技术/牌照/品牌/规模护城河 | 同质化严重，打价格战 |
| 议价能力 | 对上下游有定价权 | 被上游卡脖子、被下游压价 |
| 替代风险 | 10年内难以被替代 | 技术迭代快，随时可能被颠覆 |
| 现金流模式 | 预收款/现金结算 | 大量垫资/长账期 |

**A股好赛道：** 高端制造（半导体设备/材料）、新能源（核电/储能）、医药创新（创新药/CXO）、消费（白酒/调味品）、金融（银行/保险）

> 🔑 **行业深度展开**：
> - 银行业 [[industry-banking-framework]] — NIM驱动+资产质量+资本约束（含农行2025实战）
> - 电力行业 [[industry-power-framework]] — 火水核风四类电源+电价机制+核电投资法（含中国核电2025实战）
> - 消费行业 [[industry-consumer-framework]] — 量价拆分+渠道先行+品类扩张+品牌护城河（含海天2025实战）
> - 周期股框架 [[industry-cyclical-framework]] — 周期四阶段+先行指标+PE陷阱（火电和银行的周期属性）
> - ETF方法论 [[etf-investment-methodology]] — ETF与个股的根本区别+定投+估值择时（电力ETF 561560）
> - 医药行业 [[industry-pharma-framework]] — 创新药vs仿制药+管线估值+集采（恒瑞医药案例）
> - 科技/半导体 [[industry-tech-framework]] — 三周期叠加+科技股特化指标（CPO/光模块应用）
> - 高端制造/军工 [[industry-manufacturing-framework]] — 订单驱动+认证壁垒+国产替代+军工估值

**护城河快速对照（五选一或多选）：**
1. 无形资产（品牌/专利/牌照）→ 能不能因此提价？
2. 转换成本 → 客户想走但走不了？
3. 网络效应 → 人越多价值越大？
4. 成本优势 → 对手建同样工厂能不能追平？
5. 有效规模 → 市场只够一家吃？

### 维度二：财务健康（定量）

| 指标 | 优秀 | 及格 | 危险 |
|------|------|------|------|
| ROE | >15% | >10% | <5% |
| 营收增速(3年CAGR) | >20% | >10% | 负增长 |
| 净利润增速 | >20% | >营收增速 | 负增长或背离 |
| 资产负债率 | 30-50% | 50-70% | >80%（金融除外） |
| 经营现金流/净利润 | >1.0 | >0.8 | <0.5 |
| 应收账款/营收 | <20% | <40% | >60% |
| 商誉/净资产 | <10% | <30% | >50% |

**财务排雷四问：**
1. 利润是真的吗？（经营现金流 vs 净利润）
2. 增长可持续吗？（在手订单/合同负债/产能规划）
3. 钱去哪了？（大额在建工程/其他应收款/关联交易）
4. 大股东在做什么？（增减持/质押比例/分红政策）

> 🔑 **深度展开**：财务分析速查手册 [[financial-analysis-foundation]] — 三表勾稽 + A股造假8信号 + 银行/消费/电力行业特化指标，含农行和海天2025年报实战案例

### 维度三：估值水位

| 方法 | 怎么用 |
|------|--------|
| PE分位数 | 当前PE处于近5年什么位置？<30%分位=便宜 |
| PEG | <1=便宜，>2=贵 |
| 股息率 | >3%值得关注，>5%有安全垫 |
| PB（重资产行业） | 银行<0.8+股息率>5%=便宜 |

**A股估值口诀：**
- 银行 PB<0.8 + 股息率>5% = 便宜
- 消费 PE<25 + 增速>15% = 合理
- 科技 PE不重要，看PS+PEG+行业空间
- 周期股 PE最低的时候往往最危险（盈利顶点）

> 🔑 **深度展开**：估值方法论手册 [[valuation-methodology]] — DCF思维框架 + 行业特化估值逻辑（银行为什么看PB、消费为什么看PE）+ 反向DCF实操（当前价格在赌什么样的未来？）+ 含农行和海天当前估值实战分析

### 维度四：资金面

| 信号 | 偏多 | 偏空 |
|------|------|------|
| 北向资金 | 持续净流入 | 持续净流出 |
| 主力资金 | 超大单净流入 | 主力持续净流出 |
| 融资余额 | 温和上升 | 快速攀升后回落 |
| 股东户数 | 减少（筹码集中） | 大幅增加（筹码分散） |
| 大股东行为 | 增持/回购 | 减持/大比例质押 |

### 维度五：催化剂

| 类型 | 举例 |
|------|------|
| 业绩催化剂 | 财报超预期、新产品放量、产能投产 |
| 政策催化剂 | 行业补贴、税收优惠、监管放松 |
| 事件催化剂 | 纳入指数、重大合同、FDA/审批 |
| 周期催化剂 | 行业触底回升、库存周期反转 |
| 情绪催化剂 | 市场风格切换、板块轮动 |

> ⚠️ **没有催化剂的好公司 = 好公司但不一定是好股票**（可能长期横盘）

## 三、选股标准作业流程（SOP）

### 第一步：初筛（海选→20只）
- 财报期：利润增速>20% + ROE>15% + 经营现金流/净利润>0.8
- 日常期：北向增持 + 机构调研密集 + 底部放量
- 自上而下：确定当前市场风格（价值/成长/周期），在风格内选

### 第二步：精筛（20只→5只）
- 五维打分卡逐只评分
- 排除任意维度<3分的标的
- 优先选多维度共振（基本面好+估值低+有催化剂）

### 第三步：深度研究（5只→2-3只）
- 读最近3份财报（年报+最新季报）
- 读3份以上券商深度研报（对比不同观点）
- 查行业政策/竞争格局
- 对照已有持仓，控制行业集中度

### 第四步：制定交易计划
- 买多少？→ 单只不超过总仓位15-20%
- 什么价买？→ 支撑位/估值合理区间
- 什么价卖？→ 目标价/估值上限
- 什么情况止损？→ 基本面恶化 / -8%硬止损

## 四、分析前强制三步（每次必做）

**第一步：宏观环境扫描**
- 全球：美联储利率预期、美元指数、油价、地缘政治重大事件
- 国内：货币政策信号、A股流动性（成交额/两融/北向）、重大政策
- 美股：费城半导体指数、纳斯达克走势（对A股科技传导）
- ⚠️ **对照金融危机经典四问** [[financial-crisis-classics]]
  1. 现在处于Kindleberger五阶段的哪一阶段？
  2. 市场上最流行的"这次不一样"叙事是什么？
  3. 哪个部门的债务在快速膨胀？
  4. 如果狂热突然结束，我的持仓能不能扛住-20%到-30%？
- 🔑 利率→银行→股市的传导路径 + 持仓对宏观变量的敏感性矩阵 → [[macro-transmission-mechanism]]

**第二步：标的企业详细调查**
- 当日股价、涨跌幅、成交量、资金流向
- 近期业绩公告/重大合同/政策利好利空
- 行业板块整体表现（共振还是独立行情）
- 与手册中的买入条件、止损线的差距

**第三步：综合判断**
- 结合四位老师框架
- 给出明确操作建议（买/卖/持有/等待）+ 具体价格和时机
- 不模棱两可

## 五、交易纪律（⚠️ 最关键的一章）

### 仓位管理铁律

| 规则 | 内容 |
|------|------|
| 单只上限 | 不超过总仓位 **20%**（海天当前28.9%已超标，后续不追加） |
| 行业上限 | 单一行业不超过 **40%** |
| 现金保留 | 永远保留 **≥15%** 现金（当前31.2% ✅） |
| 分批建仓 | 任何标的至少分 **2-3次** 买入，不一把梭 |
| 首次建仓 | 不超过该标的总计划仓位的 **50%** |

### 止损纪律（不可商量）

| 级别 | 触发条件 | 行动 |
|------|---------|------|
| 🔴 硬止损 | 单只浮亏 **-8%** | 减仓一半 |
| 🔴 清仓线 | 单只浮亏 **-15%** | 全部清仓，不问原因 |
| 🟡 软止损 | 基本面恶化（业绩变脸/政策利空） | 立即评估，该走就走 |
| 🟡 时间止损 | 买入后3个月不涨反跌 | 重新评估逻辑，可能减仓 |

**当前持仓止损线速查（按6/12收盘价更新）：**
- 农行(6.80)：硬止损 6.26 / 清仓 5.78
- 海天(35.17)：硬止损 32.36 / 清仓 29.89
- 电力ETF(1.384)：硬止损 1.273 / 清仓 1.176

### 买入纪律

- ❌ 不在开盘30分钟内交易（情绪波动大）
- ❌ 不追涨停板买入
- ❌ 不在大盘暴跌日抄底（等缩量企稳）
- ✅ 在支撑位附近、缩量企稳后分批买
- ✅ 买入前必须写完交易笔记（为什么买/什么情况卖）

### 卖出纪律

- ✅ 达到目标价→ 至少卖一半
- ✅ 基本面恶化 → 不管盈亏都卖
- ✅ 发现更好的机会+现金不足 → 卖掉最弱的
- ❌ 不因"涨太多了"而卖（让利润奔跑）
- ❌ 不因"跌太多了"而卖（看止损线，不看浮亏数字）

### 心理纪律

| 禁止行为 | 原因 |
|---------|------|
| 盘中临时决定交易 | 冲动是最大的敌人 |
| 看别人赚钱就追 | 嫉妒驱动=接盘驱动 |
| 亏损后急于翻本 | 复仇交易=加倍亏损 |
| 盈利后过度自信 | 把运气当能力 |
| 不做笔记就交易 | 没有记录就没有进步 |

> 🔑 **卖出决策深度展开**：[[sell-discipline-framework]] — 三档止盈目标价+移动止盈法+基本面恶化早期信号+仓位再平衡纪律
> 🔑 **行为金融实操**：[[behavioral-finance-tools]] — 盘中自检清单+交易前后心理模板+三种常见失败模式的破解法

## 六、好股票的7个特征 & 10个危险信号

### 好股票7特征
1. 产品/服务你用过且觉得好
2. 商业模式一句话能说清楚
3. 赚的是真金白银（经营现金流持续为正且增长）
4. 管理层靠谱（承诺兑现、不乱画饼、不频繁融资稀释）
5. 行业地位稳固（前三名占大部分份额）
6. 有定价权（可以提价而不丢客户）
7. 处于成长早中期（渗透率10-50%）

### 10个危险信号（见了就跑）
1. 🔴 大股东高比例质押（>50%）+ 股价持续下跌
2. 🔴 经营现金流长期为负，靠融资续命
3. 🔴 应收账款增速远超营收增速
4. 🔴 更换审计师 / 非标审计意见
5. 🔴 大股东频繁减持 + 高管批量离职
6. 🔴 频繁跨界并购、频繁更改募资用途
7. 🔴 存货异常增长但营收不增长
8. 🔴 关联交易占比过高
9. 🔴 媒体/做空机构有实锤负面
10. 🔴 股价已涨5-10倍 + 全市场都在推荐

## 七、风险控制全景

### 组合层面
- 单只 ≤20% 仓位
- 单行业 ≤40%
- 现金 ≥15%
- 至少持有3只不相关标的（当前：银行+消费+电力 ✅）
- 每月检查一次持仓相关性

### 宏观层面
- 关注美联储议息日历（下次：**6/16-17，沃什首秀，全年最重要**）
- 关注地缘政治：美伊协议文本已达成，等待正式签署；以色列反对态度是最大变数
- 关注A股成交额（<8000亿=冷清，>2万亿=过热；6/12达3.21万亿 🔥）

### 执行层面
- 所有止损单必须提前设好（不靠盘中决策）
- 每周复盘一次交易记录
