## 一、三张报表的勾稽关系

看懂这三张表之间的关系，比看懂单张表的数字重要得多。造假最容易露馅的地方，就是三张表之间的逻辑对不上。

### 1.1 利润表 → 资产负债表

**核心公式**：
```
期末未分配利润 = 期初未分配利润 + 本期净利润 − 分红
```

**实操意义**：如果净利润连续增长但净资产不增长（或增长很慢），说明钱被分掉了或者被"其他"渠道消耗了。

**海天 2025 案例**：
```
2025 年归母净利润 70.38 亿，同比增长 10.95%
但 ROE 从 21.77% 降到 19.59%
→ 为什么利润涨、ROE 跌？
→ 因为 H 股上市导致净资产增加了 33.8%（分母变大 > 分子增长）
→ 这不是经营恶化，是股权融资的后果——钱多了但还没产生收益
```

**农行 2025 案例**：
```
净利润 2910 亿（+3.2%）→ 留存利润增加净资产
但银行不能只看 ROE，要看 RORWA（风险加权资产回报率）
→ 因为银行的核心约束是资本充足率
```

### 1.2 利润表 → 现金流量表

**这是最重要的勾稽关系。** 利润是"权责发生制"下的数字，现金流是"真金白银"。两者的差距告诉你利润的质量。

**核心公式**：
```
经营活动现金流净额 ≈ 净利润 + 非现金费用（折旧/摊销）− 营运资金增加
```

**实操判断**：
| 经营现金流/净利润 | 含义 |
|------------------|------|
| > 1.0 | 利润质量好，赚的是真钱 |
| 0.8-1.0 | 正常范围 |
| 0.5-0.8 | 利润回收偏慢，关注应收/存货 |
| < 0.5 | 危险——利润可能只是账面上的 |
| 长期为负 | 大概率在烧钱或造假 |

**海天 2025**：海天是典型的消费品公司，经营现金流通常 ≥ 净利润（预收款模式 + 低资本开支），这是优质消费股的财务特征。

**农行 2025**：银行的现金流表含义不同——贷款放出是"流出"，存款增加是"流入"，不能直接用上面的比率。银行的"利润质量"看的是不良认定严格度和拨备充足度。

### 1.3 三表联动的三个关键信号

| 信号 | 利润表 | 资产负债表 | 现金流量表 | 含义 |
|------|--------|-----------|-----------|------|
| **虚增收入** | 收入↑ | 应收款↑↑ | 经营现金流不跟涨 | 放宽信用政策冲收入，客户可能没真给钱 |
| **虚增利润** | 利润↑ | 存货↑↑ / 在建工程↑ | 经营现金流不跟涨 | 产能过剩继续生产摊薄成本，或延迟转固少提折旧 |
| **利润真实** | 利润↑ | 应收温和增长 | 经营现金流同步增长 | ✅ 健康的增长 |

## 二、A 股财务造假高频信号

以下信号出现任何一个都要警惕；出现两个以上，基本可以放弃了。**你不需要证明造假，只需要避开有嫌疑的。**

### 信号 1：存贷双高
```
账上有巨额现金 + 同时大额借款
→ 如果真有钱，为什么要借？
→ 可能：现金是假的（被大股东占用，审计后才转回）
```

### 信号 2：应收账款增速远超营收增速
```
营收 +10%，应收款 +50%
→ 货是卖出去了，但客户没给钱
→ 可能是：放宽信用政策冲业绩，或者直接虚构销售
```

**农行不适用**（银行没有"应收款"这个概念），但消费/制造业必须看这个。

### 信号 3：存货异常增长但营收不增长
```
存货 +40%，营收 +5%
→ 货生产出来了但卖不出去
→ 也可能是：不减值、不处理，虚增资产和利润
```

### 信号 4：在建工程长期不转固
```
在建工程持续几年不转入固定资产
→ 转固就要开始提折旧 → 减少利润
→ 所以故意拖着不转 → 虚增利润
```

### 信号 5：大股东高比例质押 + 减持
```
大股东质押 > 50% + 持续减持
→ 股东自己在跑
→ 质押比例过高意味着爆仓风险——股价一跌，券商强平，进一步砸盘
```

### 信号 6：商誉占净资产过高
```
商誉/净资产 > 50%
→ 前面做了高溢价并购
→ 如果被收购公司业绩不达预期 → 商誉减值 → 一次性巨额亏损
```

### 信号 7：频繁更换审计机构
```
三年换了两次审计师
→ 审计师发现了问题不愿配合
→ 或者公司"炒掉"了不合作的审计师
→ 非常强烈的危险信号
```

### 信号 8：非标审计意见
```
审计报告是"保留意见""无法表示意见""否定意见"
→ 审计师说"这账我看不懂"或"有问题"
→ ST 前兆，直接否决
```

### 🔍 快速筛查清单（拿到财报 5 分钟跑完）

```
□ 经营现金流/净利润 > 0.8？（银行除外）
□ 应收账款增速 < 营收增速？
□ 存货增速 < 营收增速？
□ 无在建工程长期不转固？
□ 大股东质押 < 50% 且无大规模减持？
□ 商誉/净资产 < 30%？
□ 审计意见是"标准无保留"？
□ 近三年没有频繁换审计师？
```

**8 项全过的，财务造假风险较低。** 任何一项不过，都要追问原因。

## 三、行业特化指标

### 3.1 银行业（以农行为例）

银行的报表和普通企业完全不同——"原材料"是存款，"产品"是贷款，"价格"是利率。不能用 PE/应收账款/存货那一套。

#### 核心指标四件套

```
① 净息差（NIM）= 生息资产收益率 − 计息负债成本率
② 不良贷款率 + 关注类贷款迁徙率
③ 拨备覆盖率 = 贷款损失准备 / 不良贷款余额
④ 资本充足率（核心一级 / 一级 / 总）
```

**农行 2025 实战分析**：

| 指标 | 数值 | 解读 |
|------|------|------|
| 净息差 | 1.28%（↓14bp） | 历史低位，但降幅逐季收窄，Q4 仅降 14bp → 企稳信号 |
| 不良率 | 1.27%（↓3bp） | 持续改善 |
| 拨备覆盖率 | 292.55% | 极高水平，相当于"藏了"近 2 倍利润可以在未来释放 |
| 核心一级资本充足率 | 10.96% | 高于监管要求（8.5%），但不算宽裕 |

**四指标联动判断**：
- 净息差下降 + 拨备覆盖率极高 → 银行在用拨备"平滑"利润。不披露的真相可能是息差压力被拨备缓冲了
- 不良率下降 + 拨备覆盖率也下降 → 两种可能：①真的在改善（不良双降）②通过核销/处置把不良"做没了"
- 判断方法：看**不良生成率**。农行 2025 为 0.57%（2024 为 0.60%）→ 真实改善

**银行 ROE 拆解（杜邦分析）**：
```
ROE = 净利率 × 资产周转率 × 权益乘数
    ≈ (净利润/总收入) × (总收入/总资产) × (总资产/净资产)

农行 2025 ROE = 10.16%
→ 这不是消费品那种"轻资产高利润"的 ROE
→ 是靠权益乘数（杠杆）撑起来的
→ 所以资本充足率直接决定了增长天花板
```

**银行股的投资逻辑**：不追求高增长（贷款增速受制于资本），追求的是**低 PB + 高股息 + 资产质量改善**。不良率拐点才是银行股最大的催化剂。

### 3.2 消费/食品行业（以海天为例）

消费股是 A 股财务最干净的板块之一——预收款、高现金流、低应收。但这不代表没有盲区。

#### 核心指标

```
① 毛利率 = 产品有没有定价权
② 合同负债（预收款）= 经销商抢不抢货（先行指标）
③ 销售费用率 = 增长是靠品牌还是靠砸钱
④ 经销商单产 = 渠道效率
⑤ ROE 及杜邦拆解
```

**海天 2025 实战分析**：

| 指标 | 数值 | 解读 |
|------|------|------|
| 毛利率 | 40.22%（+3.22pct） | 原料成本下降 + 灯塔工厂降本 → 毛利率终于回到 40%+ |
| 合同负债 | 41.3 亿（↓2.1 亿） | 经销商打款意愿略有下降，但降幅不大 |
| ROE | 19.59%（↓2.18pct） | H 股上市摊薄，不是经营恶化 |
| 净利率 | 24.40% | 极致水平，说明几乎没有真正的竞争对手 |

**海天 ROE 杜邦拆解**：
```
ROE = 净利率 24.40% × 资产周转率 × 权益乘数
→ 海天的 ROE 靠的是净利率（产品强）+ 周转率（动销快）
→ 杠杆很低——不需要借钱经营
→ 这是最好的 ROE 结构
```

**消费股的关键问题**：增长来自量还是价？

```
海天酱油 149 亿（+8.55%）→ 量还是价？
如果量的增速 < 营收增速 → 靠提价 → 提价有天花板
如果量的增速 > 营收增速 → 渗透率在提升 → 更可持续

海天年报里的答案：
→ 酱油增速中，量增和结构升级各占约一半
→ 品类扩展（其他 +14.55%）才是更大的增长来源
→ 所以看海天，核心不是酱油提价，是"其他品类能不能接棒"
```

**经销商是先行指标**：
```
经销商数量微降（-5 家）→ 不是大规模汰换，渠道稳定
但合同负债减少 2.1 亿 → 经销商打款节奏变慢
→ 需要关注 Q1 数据：如果合同负债继续减少，说明动销在变慢
```

### 3.3 电力/公用事业

这部分放在独立的 [[industry-power-framework]] 中详细展开（阶段 3），这里只写最核心的逻辑：

**与其他行业最大的区别**：
- 利润不是"卖得多就赚得多"——电价大概率是管制的，量增长受装机容量上限约束
- 看电力公司，看的是三个东西：**上网电量 × 上网电价 − 成本 = 利润**
- 成长性来自**装机增长**（新建机组），不是来自同店增长
- 重资产属性：折旧占成本大头，经营性现金流远超净利润（折旧是账面费用、不影响现金流）
- **股息率是最核心的估值锚**——因为增长有限，你能拿到的就是分红

**火电、水电、核电、风电的关键变量各不相同**，详见阶段 3。

## 五、实用速查

### 拿到一份 A 股年报，30 分钟要做什么

```
⏱ 0-5 分钟：快速筛查
  → 跑完造假信号 8 问
  → 任何一个不过 → 追问原因

⏱ 5-15 分钟：核心数据
  → ROE 及杜邦拆解（净利率 × 周转率 × 杠杆）
  → 营收增速、利润增速是否匹配
  → 经营现金流/净利润

⏱ 15-25 分钟：行业特化
  → 银行：NIM + 不良 + 拨备 + 资本
  → 消费：毛利率 + 合同负债 + 经销商 + 量价拆分
  → 电力：利用小时数 + 度电利润 + 装机规划 + 股息

⏱ 25-30 分钟：综合判断
  → 财务健康程度（好/一般/危险）
  → 跟去年比变好了还是变差了
  → 最大的风险点是什么
```

---

**Why:** 用户现有投资体系中财务分析维度偏弱——有指标但没有理解指标背后的故事。本手册以实际持仓（农行、海天）2025 年报为案例，把三表勾稽、造假信号、行业特化指标整合到一份速查手册中。

**How to apply:** 每次分析新标的时，先跑完第二章的快速筛查清单排除造假嫌疑，再用第三章的行业特化指标做深入分析。本手册是 [[investment-handbook]] 维度二的深度展开。
