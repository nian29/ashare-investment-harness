# 季报/年报速读模板

> 一份季报几十页，大部分内容你不用看。这个模板告诉你**重点看哪几行、数字变了代表什么、30分钟内跑完**。

---

## 第一步：先搞清楚你在看什么（1分钟）

```
□ 这是第几季度的报告？
  → Q1（1-3月）、半年报（1-6月）、Q3（1-9月）、年报（全年）
  → 季报信息少（只有基本财务数据），年报信息多（有详细的经营讨论）

□ 跟前一个季度比，还是跟去年同一个季度比？
  → 例：2026年Q1 vs 2025年Q1 = 跟去年同季度比（可比）
  → 例：2026年Q1 vs 2025年Q4 = 跟前一季度比（不太可比，有季节性）
```

---

## 第二步：看"摘要"——三个最关键的硬数字（5分钟）

打开年报/季报，直接翻到**财务数据摘要**。就盯这四个数：

### 硬数字 1：营业收入（卖了多少货/收了多少费）

```
跟去年同季度比：
  ↑ 超过10%：增长强劲
  ↑ 0-10%：增长一般
  ↓ 下降：警惕——是行业问题还是公司问题？
```

### 硬数字 2：归母净利润（扣掉所有成本和税之后，真正赚的钱）

```
跟去年同季度比：
  ↑ 超过营收增速 → 利润在加速（好事）
  ↑ 但慢于营收增速 → 成本涨得比收入快（警惕）
  ↓ 但营收在增长 → 成本失控或一次性损失
  ↓ + 营收也在降 → 公司出问题了
```

### 硬数字 3：扣非净利润（去掉一次性意外之财/灾之后的利润）

```
归母净利润 跟 扣非净利润 的差距：
  归母 >> 扣非 → 利润有水分（卖资产、政府补贴等一次性收入）
  归母 ≈ 扣非 → 利润是实打实赚的 ✅
  
  例：归母净利10亿，扣非净利6亿 → 4亿是靠卖楼赚的，不是主业
```

### 硬数字 4：经营现金流净额（真金白银进了多少）

```
经营现金流 vs 净利润：
  经营现金流 > 净利润 → 利润是真金白银 ✅
  经营现金流 < 净利润（连续两个季度）→ 利润是应收账款（纸面富贵）⚠️
  经营现金流为负 → 赚了利润但兜里更空了 🔴
```

---

## 第三步：查危险信号——A股最常见的问题（10分钟）

### 信号 1：应收账款增速 > 营业收入增速

```
什么意思：货卖了，但客户不给钱，只给了一张欠条。
         营收涨10%，应收涨30% → 为了冲业绩在放水，客户可能还不起。

怎么看：年报里搜"应收账款"，对比期末数和期初数的变化。
        季报数据不如年报全，但趋势能看到。

绿灯：应收增速 < 营收增速
黄灯：应收增速 > 营收增速（单季度）
红灯：连续两个季度应收增速 > 营收增速
```

### 信号 2：存货大增但营收不增

```
什么意思：生产了一堆货，但卖不出去，堆在仓库里。

怎么看：搜"存货"，对比变化。

绿灯：存货增速 < 营收增速（生产跟得上销售）
黄灯：存货增速 > 营收增速（单季度）
红灯：存货增速 > 20% + 营收下降（严重的滞销）
```

### 信号 3：毛利率连续下滑

```
什么意思：产品卖不出好价钱了，或者成本控制不住了。

怎么看：利润表里，毛利率 = (营收 - 营业成本) ÷ 营收。

绿灯：毛利率稳定或上升
黄灯：毛利率连续两个季度下降（累计降幅 < 2个百分点）
红灯：毛利率连续两个季度下降（累计降幅 > 3个百分点）
     → 竞争加剧了，或者原材料大涨
```

### 信号 4：经营现金流变脸

```
什么意思：利润表好看，但兜里没钱进来。

怎么看：现金流量表第一行"经营活动产生的现金流量净额"。

绿灯：经营现金流为正 + 接近或大于净利润
黄灯：经营现金流为正但大幅低于净利润
红灯：经营现金流为负（连续两个季度）
     → 公司可能在给客户"赊销"，利润是纸面的
```

### 信号 5：大股东质押或减持

```
什么意思：最大的股东在偷偷逃跑。

怎么看：年报/季报的"股东情况"章节，看控股股东质押比例。

绿灯：质押比例 < 30%
黄灯：质押比例 30-50%
红灯：质押比例 > 50% + 最近在减持
     → 大股东缺钱，而且对公司没信心
```

## 第四步：分行业专项检查——你的持仓专属（10分钟）

### 🏦 银行股（农行）专属

```
□ 净息差（NIM）：跟上个季度比是升还是降？
  → 降了 = 赚的利差变薄了（利率下行环境中这是行业性问题，不是农行一家的事）

□ 不良贷款率：比上季度升了吗？
  → 升了 > 0.05个百分点 = 坏账在增加

□ 拨备覆盖率：这个数够不够？
  → > 200% = 安全垫很厚
  → < 150% = 注意

□ 核心一级资本充足率：
  → > 10% = 健康
  → < 8.5% = 可能需要再融资（稀释你的股权）
```

### 🧧 消费股（海天）专属

```
□ 合同负债（旧称预收款）：比上个季度变化？
  → 增加 = 经销商在积极打款，后续销售有保障 ✅
  → 减少 > 10% = 经销商拿货意愿下降 ⚠️
  → 连续两个季度减少 = 渠道出问题了 🔴

□ 经销商数量：年报里搜"经销商"，数量在增还是减？
  → 增加 = 渠道在扩张
  → 减少 = 渠道在萎缩（非常危险）

□ 其他品类增速（蚝油、调味酱、复合调味料）：
  → 维持 > 10% = 品类扩张逻辑在兑现 ✅
  → 跌破 10% = 增长遇到瓶颈
```

### ⚡ 电力股（电力ETF/核电）专属

```
□ 对于ETF：看前十持仓的变化 + 整体净值
  → ETF的核心是行业景气度，不用逐只盯

□ 对于中国核电：装机容量变化 + 发电小时数
  → 年报里"管理层讨论"章节有详细数据

□ 煤价走势（影响火电利润）：
  → 煤价跌 = 火电利润增
  → 煤价涨 = 火电利润承压
```

### 🛣️ 高速公路股（宁沪高速）专属

```
□ 各路段日均车流量：公司每月有经营数据公告
  → 连续增长 = 正常
  → 某条路突然大降 = 可能被新路分流

□ 毛利率变化：
  → 高速公司毛利率通常稳定（50-65%）
  → 如果大幅波动，去看是不是改扩建导致成本增加

□ 改扩建进度（年报"管理层讨论"章节）：
  → 是否按计划推进？
  → 有没有延期或超预算？

□ 资本开支：比去年同期增加还是减少？
  → 增加 = 在扩建，短期压制现金流
  → 减少 = 扩建高峰过去了，现金流好转
```

## 第五步：看"管理层讨论"——公司怎么说（5分钟）

年报里有一章叫"管理层讨论与分析"（季报里叫"经营情况讨论"），这是**公司用自己的话解释数字背后发生了什么**。

```
重点看这几个段落：
① "报告期内公司经营情况" → 公司自己承认的问题是什么？
② "行业格局和趋势" → 公司对行业的判断，跟你判断的一致吗？
③ "风险因素" → 公司承认的风险，跟你想的风险一样吗？

实用技巧：
  → 管理层讨论跟数字对得上 → 公司在诚实地解释
  → 数字差了但管理层避而不谈 → 警惕
  → 数字好了就大吹，数字差了就甩锅给"宏观环境" → 管理层不诚实
```

---

## 第六步：30分钟速读总流程

```
[0-1分钟]  搞清楚是哪份报告，跟谁比
[1-5分钟]  扫四个硬数字：营收、归母净利、扣非净利、经营现金流
[5-15分钟] 查五个危险信号：应收、存货、毛利率、现金流、质押
[15-25分钟] 行业专项检查（根据你的持仓选）
[25-30分钟] 读管理层讨论 + 形成判断

30分钟后你应该能回答这三个问题：
  □ 公司这个季度比去年同季度是好是坏？
  □ 有没有触发任何危险信号？
  □ 我对这家公司的判断需要更新吗？（贝叶斯更新）
```

**Why:** 用户持有4只标的（农行、海天、电力ETF、宁沪高速），每次季报出来需要系统化地检查。这个模板把30分钟能跑完的流程压缩成了可操作的步骤，避免"看了半天不知道重点在哪"。

**How to apply:** 季报/年报出来后，打开本模板，按六步流程跑一遍。跑完后在投资日报里写一条"季报检查"记录。如果触发任何红灯信号，加载[[sell-discipline-framework]]判断是否需要减仓。
