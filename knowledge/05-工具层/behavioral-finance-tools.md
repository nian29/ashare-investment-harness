# 行为金融实操工具

> 在 [[poor-charlies-almanack]] 的 25 种误判心理学基础上，加一套可以在交易中实际使用的方法。
> 知识不变成工具，就等于没有知识。

---

## 一、25 种偏误 → 5 个交易中最容易犯的

不用记住全部 25 种。在你现在的投资阶段，这 5 个是最容易出问题的：

| 偏误 | 你的交易中怎么出现 | 芒格编号 |
|------|------------------|---------|
| **从众/社会认同** | "CPO 涨停了全市场都在买，我也应该买" | #15 |
| **被剥夺超级反应** | "浮亏 10% 了，我受不了，割肉" → 割在最低点 | #14 |
| **避免不一致性** | "我已经研究过了这家好，跌了也不改结论" → 拒绝承认错误 | #5 |
| **自视过高** | "这次我肯定看对了" → 过度集中、不上止损 | #12 |
| **激励倾向** | "涨了 30% 我太厉害了" → 把运气当能力，下次下重注 | #1 |

## 二、盘中自检清单——在你想交易的那一刻

当你产生一个强烈的交易冲动时（想买 or 想卖），停 3 分钟，跑这个清单：

### 想买入时

```
□ 我是因为涨了才想买吗？
  → 是 = 从众偏误 + 害怕错过（FOMO）
  → 不是，是因为基本面变化 → 可以继续评估

□ 这个想法是盘中产生的还是计划好的？
  → 盘中产生 → 很可能是情绪驱动，不能执行
  → 计划好的 → 可以执行（但也要检查下面几条）

□ 我现在是不是在嫉妒（看别人赚钱）？
  → 是 → 停下来。嫉妒驱动的买入没有一次是好的

□ 我能不能在两分钟内跟 10 岁小孩讲清楚这公司怎么赚钱？
  → 不能 → 能力圈之外，不买

□ 如果明天大盘跌 5%，我还想买吗？
  → 不想 → 你是在赌方向，不是在投资
```

### 想卖出时

```
□ 是因为跌了才想卖吗？
  → 是 → 被剥夺超级反应。触发止损线了吗？
    → 没有触发 → 不能卖。这只是情绪波动
    → 触发了 → 按纪律执行

□ 是因为涨了很多才想卖吗？
  → 是 → 锚定效应（觉得"赚够了"）
  → 到止盈目标价了吗？到了 → 按计划卖。没到 → 移动止盈说了算

□ 是因为别人都在卖？
  → 是 → 从众偏误。你是因为他们卖才想卖，还是因为基本面变了？

□ 基本面和买入时比变了吗？
  → 没变 → 不要卖（除非到止盈目标价）
  → 变了 → 是暂时问题还是结构问题？→ 见 [[sell-discipline-framework]] 决策树
```

## 三、交易前心理检查——买入前必须答完

这是你要写在交易笔记里的，不是脑子里过一遍就算。

```
1. 我现在的情绪状态是什么？
   冷静 / 兴奋 / 焦虑 / 害怕错过 / 复仇（之前亏了想赚回来）

2. 如果答案是"冷静"以外的任何选项 → 今天不交易。

3. 这笔交易如果完全亏光了，我的生活会受影响吗？
   是 → 仓位太重，减半
   否 → 可以继续

4. 我是否觉得自己"这次肯定对了"？
   是 → 自视过高偏误。重新检查反对理由

5. 如果明天市场就证明我错了（跌 8%），我能接受吗？
   不能 → 仓位太重或心态没准备好
   能 → 可以执行
```

**铁律**：如果你在"兴奋"或"焦虑"状态下做的任何交易决定，90% 是错的。在你感觉"什么都不该做"的时候做的决定，往往是对的。

## 四、交易后复盘模板——每笔交易都要填

不是日记，是模板。用结构化问题替代自由发挥。

```
━━━━━━━━━━━━━━━━━━━━━━━━
交易复盘 | 日期：____ 标的：____ 方向：买/卖
━━━━━━━━━━━━━━━━━━━━━━━━

1. 这笔交易的逻辑是什么？（一句话）
   _________________________________

2. 这是计划内的还是临时的？
   □ 计划内（提前至少一天想好的）
   □ 临时（盘中决定）

3. 交易时我的情绪状态？
   □ 冷静  □ 略紧张  □ 兴奋  □ 焦虑  □ FOMO

4. 回头看，最可能出错的是什么？
   _________________________________

5. 这笔交易里，我是对的因为运气还是能力？
   _________________________________

6. 有没有我能改进的？
   _________________________________

7. 如果是亏损交易：
   □ 是纪律问题（没按止损/止盈执行）→ 这是唯一不可原谅的错误
   □ 是判断问题（逻辑错了）→ 修正逻辑
   □ 是运气问题（逻辑对但随机事件干扰）→ 没关系，不要改

━━━━━━━━━━━━━━━━━━━━━━━━
```

**七问里，第三问和第七问最重要。** 情绪记录帮你发现自己的偏误模式，亏损分类帮你区分"需要改的错误"和"不需要改的运气"。

## 五、三种最常见的失败模式——提前知道怎么输

### 模式 1：追高死在最高点

```
剧本：
1. 看到全市场都在推荐某个板块（如 CPO）
2. 忍了几天没买 → 板块继续涨 → 每天都在错过的痛苦中
3. 终于忍不住买进去了 → 第二天板块转跌
4. 跌了 5% → "这只是正常回调" → 继续拿
5. 跌了 15% → "现在卖太亏了，等反弹回本就卖"
6. 跌了 30% → "完了，下不了手了"

偏误链条：社会认同(#15) → 害怕错过 → 避免不一致(#5) → 被剥夺超级反应(#14)

破解：永远不在"大家都在推荐"的时候买入。市场最热的时候，写一条笔记："如果它真的这么好，为什么这么着急现在买？"
```

### 模式 2：在最低点割肉

```
剧本：
1. 做了充分研究后买入
2. 买入后继续跌 10%
3. 每天看着浮亏数字睡不着
4. 某天大盘暴跌，你的持仓又跌了 5%
5. "我受不了了，全部清仓"
6. 两周后，反弹了 20%

偏误链条：被剥夺超级反应(#14) → 恐惧 → 从众(#15)

破解：止损线是提前设的，不是盘中临时决定的。只要没到止损线，就别看股价。每次想割肉时，重新跑一遍原始买入逻辑——如果逻辑没变，就不卖。
```

### 模式 3：让利润坐过山车

```
剧本：
1. 在低位买入 → 涨了 40%
2. "还会涨的，我长期看好"
3. 最高点没卖（不知道那是最高点）
4. 跌了 10% → "正常回调"
5. 跌了 25% → "现在已经比最高点低这么多，等反弹再卖"
6. 最终卖掉时，只剩下 10% 利润

偏误链条：自视过高(#12) → 避免不一致(#5) → 禀赋效应

破解：移动止盈。不要判断"哪里是顶"，让规则替你判断。从最高点回撤 8%，无条件减仓一半。
```

## 六、三个保护机制

### 机制 1：冷却期

```
任何盘中产生的交易冲动 → 必须冷却至少 2 小时才能执行
如果是下午 14:30 以后才产生的冲动 → 明天开盘后 30 分钟以后再说
（避免尾盘情绪交易 + 开盘恐慌交易）
```

### 机制 2：交易笔记前置

```
买入前必须写完为什么买、什么情况卖、目标价、止损价
→ 写完才能下单
→ 不能先买了再补笔记

这条是你在手册里写的，但执行率怎么样？
```

### 机制 3：他人视角

```
"如果我最好的朋友问我该不该买/卖这只股票，我会怎么回答？"
→ 换成"给朋友建议"的视角，你的回答会理性得多
→ 因为我们给别人建议时，会自动过滤掉情绪成分
```

## 七、跟老巴和芒格学一句话

> "投资不是智商游戏。智商 160 的人打败不了智商 130 的人。你需要的是控制冲动的能力，那些冲动会给别人带来麻烦。" ——巴菲特

你的金融知识和投资框架已经够用了。剩下最重要的差距不在知识，在**能不能在自己最想交易的时候不交易**。

---

**Why:** 芒格的 25 种偏误是知识，但没有变成工具。本手册把最相关的 5 种偏误映射到交易场景，提供盘中自检、交易前后心理模板、三种失败模式的破解法。

**How to apply:** 每次产生交易冲动时跑盘中自检清单（2-3 分钟）。每笔交易后填复盘模板（5 分钟）。坚持一个月，你会发现自己的偏误模式。
