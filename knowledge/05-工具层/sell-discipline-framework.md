# 卖出决策框架

> 配合 [[investment-handbook]] 第五部分（交易纪律）使用。
> "知道什么时候买，不知道什么时候卖"是绝大多数投资者最大的亏损来源。本框架把卖出规则系统化。

---

## 一、为什么卖出比买入更难

**买入时**：你在做研究、跑五维、写笔记 → 理性主导
**卖出时**：你在看浮盈/浮亏、被贪婪/恐惧支配 → 情绪主导

三个最致命的卖出心理陷阱：

| 陷阱 | 表现 | 后果 |
|------|------|------|
| **禀赋效应** | "我买的当然好，跌了只是暂时的" | 拒绝止损，小亏变大亏 |
| **处置效应** | "卖掉亏损的 = 承认错误" | 死扛亏损股，卖飞利润股 |
| **锚定效应** | "我成本 35 块，必须回到 35 才卖" | 成本价不应该是卖出决策的任何参考 |

**核心原则**：**卖不卖跟你的买入价无关。只跟"现在的基本面+估值+仓位"有关。**

## 二、止盈框架——"什么时候该走了"

现有手册只有止损（-8%/-15%），止盈规则是空的。补上。

### 2.1 三档目标价法

买入前就要设三档目标价：

| 档位 | PE/PB 假设 | 触发后操作 | 逻辑 |
|------|----------|----------|------|
| **保守目标** | 历史估值中枢上沿 | 卖出 **1/3** | 锁定一部分利润，降低心理压力 |
| **基准目标** | 乐观但不疯狂的估值 | 再卖 **1/3** | 此时大部分利润已兑现 |
| **乐观目标** | 景气巅峰估值 | 清仓剩余 | 不要想卖在最高点 |

**海天实战示例**：
```
当前 PE：26.78 倍（2026.6.17）

假设你买入时 PE = 25 倍（历史中枢下沿）
保守目标：PE 30 倍 → 涨幅 ~20% → 卖出 1/3
基准目标：PE 35 倍 → 涨幅 ~40% → 再卖 1/3
乐观目标：PE 40 倍 → 涨幅 ~60% → 清仓

⚠️ 目标价是动态的——如果海天 2026 年底净利润增长 12%，目标价也同步上调 12%
```

**农行实战示例**：
```
当前 PB：0.81

农行看 PB，不看 PE
保守目标：PB 0.9 → 涨幅 ~11% → 卖出 1/3
基准目标：PB 1.0 → 涨幅 ~23% → 再卖 1/3
乐观目标：PB 1.1 → 涨幅 ~36% → 清仓
（国有大行 PB 极少超过 1.2，设定 1.1 是极其乐观的假设）
```

### 2.2 移动止盈法

三档目标价是"计划"，但市场可能不到目标价就掉头。用移动止盈来保护利润。

```
规则：从最高点回撤 -8%，触发移动止盈

操作：
→ 记录持仓以来的最高价（每日收盘为准）
→ 如果当前价 ≤ 最高价 × 92% → 减仓一半
→ 如果从减仓价继续跌 -8% → 清仓剩余

案例：
海天涨到 40 元（历史最高）
→ 跌到 40 × 0.92 = 36.8 → 卖一半
→ 继续跌到 36.8 × 0.92 = 33.86 → 全卖
→ 你锁定了 36.8-40 区间的绝大部分利润
```

**为什么用 -8% 而不是 -5% 或 -15%？**
- -5%：太窄，正常波动就会触发，频繁交易
- -15%：太宽，等你反应过来已经回吐太多
- -8%：跟硬止损一致，执行起来不需要纠结"这是止盈还是止损"

### 2.3 估值驱动的止盈

如果标的 PE 已经远超历史中枢，不需要等移动止盈触发，应该主动减仓。

```
触发条件：PE 超过历史 90% 分位
操作：先卖一半，剩下的用移动止盈保护

海天：如果 PE 涨到 45 倍（历史 85% 分位）
→ 先卖一半（估值已经到了市场乐观的边缘）
→ 剩下的一半用移动止盈（万一市场更疯，还能继续赚）
```

## 三、基本面恶化早期信号——"不等季报就知道变差了"

等季报出来才知道变差，股价已经跌了 20%。需要在季报之前就嗅到味道。

### 3.1 季报前预警信号（不需要等财报）

| 信号 | 去哪看 | 含义 |
|------|--------|------|
| **行业数据连续下滑** | 行业网站、统计局 | 大环境变差，单家公司很难独善其身 |
| **频繁更换经销商** | 公司公告、行业群 | 渠道出问题，可能是压货压不动了 |
| **产品降价促销** | 电商平台、超市 | 动销变慢，需要降价清库存 |
| **核心人员离职** | 公司公告 | 内部出了问题 |
| **大股东频繁减持** | 大宗交易公告 | "内部人"在不看好 |
| **行业竞品数据下滑** | 竞品季报 | 行业问题而非公司问题 |

**海天具体**：关注调味品行业出货量（统计局/行业协会），如果连续两个月下滑但海天还在喊增长 → 警惕。

**农行具体**：关注 LPR（贷款基准利率）调整和信贷数据。LPR 下调 → 净息差进一步压缩。

**核电具体**：关注电力市场化交易电价的月度数据。如果电价连续几个月在降 → 提醒自己"这个季度的利润可能不好看"。

### 3.2 季报中的恶化信号（出来后快速判断）

| 信号 | 严重程度 | 出现后做什么 |
|------|---------|------------|
| 应收款增速 > 营收增速（连续两季度） | 🔴 高 | 减仓 50%，等解释 |
| 毛利率连续两季度下降 | 🔴 高 | 确认原因（是原料涨还是被迫降价？）降价 → 清仓 |
| 经营现金流/净利润 < 0.5 | 🔴 高 | 利润在变成纸，减仓 |
| 营收增速连续两季度 < 5%（消费品） | 🟡 中 | 重新评估增长逻辑 |
| 合同负债连续两季度下降 > 10% | 🟡 中 | 经销商在撤退，密切关注 |
| 分红被砍（银行） | 🔴 高 | 立刻重新评估。银行砍分红 = 资本出问题或者利润出问题 |

### 3.3 基本面恶化的决策树

```
信号触发
  ├── 是一个短期噪声？（如天气影响、一次性费用）
  │     → 观察，不做操作
  │
  ├── 是一个行业周期性问题？（如煤价涨、利率降）
  │     → 评估持仓比例，考虑减仓但不一定清仓
  │
  └── 是公司自身的结构性问题？（如市占率下降、产品被替代）
        → 不管盈亏，清仓。这不是暂时的。
```

**关键区分**：行业周期问题是"蹲下后会站起来"，公司结构问题是"再也站不起来了"。

## 四、仓位再平衡纪律

### 4.1 仓位超标自动减仓

```
规则：
□ 单只超过 25% 总仓位 → 强制减到 20%（海天之前 28.9%，触发此规则）
□ 单行业超过 40% → 强制减到 35%
□ 现金低于 10% → 选择最贵或最弱的持仓卖出，回到 15% 现金
```

**注意**：仓位超标不是因为"涨多了"——是因为它在组合里占比太大了。哪怕你觉得它还会涨，也要减——这是风险控制，跟判断无关。

### 4.2 牛市中的"割韭菜"纪律

牛市中最大的错误是"一直拿着不卖"。你以为自己在让利润奔跑，实际上是在让利润坐过山车。

```
每季度检查一次：
□ 有没有持仓浮盈 > 30%？
  → 卖 1/3，把本金拿回来一部分
  → 剩下的让利润跑（用移动止盈保护）

□ 有没有持仓浮盈 > 50%？
  → 至少卖一半
  → 如果市场整体偏热（成交 > 2 万亿），卖 2/3

道理很简单：
  → 浮盈不是你的钱，卖掉后的现金才是
  → "卖飞了"的痛苦 << "坐了过山车又跌回去"的痛苦
  → 你永远不会因为落袋为安而亏钱
```

### 4.3 调仓纪律——卖掉最弱的

当需要现金（比如看到更好的机会，或者需要补充现金储备），卖什么？

```
不是卖"涨得最多的"，是卖"逻辑最弱的"。

排序方法：
1. 基本面恶化最明显的
2. 估值最贵的（相比行业内）
3. 催化剂最远的
4. 护城河最窄的

按这个顺序，从最差的开始卖。
```

## 五、卖出的五个"绝不"

1. **绝不因"涨太多了"而匆忙卖。** 涨多了不是卖的理由，涨到超过合理估值上限才是。让利润奔跑，但用移动止盈绳拴着。
2. **绝不因"跌太多了"而恐慌卖。** 跌到了止损线才卖，不是"跌到你觉得难受"就卖。
3. **绝不因"别人都在卖"而卖。** 这是从众，不是决策。
4. **绝不在盘中临时决定卖。** 所有卖出条件都是提前设好的。
5. **绝不因为"找到了更好的机会"而匆忙卖。** 先研究新机会，确认它确实更好，再执行调仓。

## 六、与现有手册的对接

本框架是 [[investment-handbook]] 第五部分（交易纪律）的深度展开。使用顺序：

1. 买入任何标的时 → 同时写好三档目标价 + 移动止盈触发价
2. 持有期间 → 每季度检查基本面恶化早期信号
3. 触发卖出条件 → 执行，不等"更好的机会"或"感觉还能涨"

**把卖出条件写在买入笔记里**，这是最重要的一条纪律。

---

**Why:** 用户现有体系买入规则完善但卖出规则薄——只有止损没有止盈，没有基本面恶化的早期识别。本框架补充了止盈三档法、移动止盈、早期预警信号、仓位再平衡四条线，形成完整的"买→持→卖"全生命周期规则。

**How to apply:** 每次买入后立即设定三档止盈目标价和移动止盈触发价。每季度对照基本面恶化信号做持仓体检。触发任何卖出条件就执行。
