# 估值方法论

> 配合 [[investment-handbook]] 维度三（估值水位）使用。
> 马克斯说"买得好比买好的更重要"——但你怎么知道什么是"好价格"？这篇回答这个问题。

---

## 一、估值的第一性原理

### 任何资产的价值 = 未来能产生的所有现金流的现值

这不是一句空话，是估值唯一的底层逻辑。所有方法（PE、PB、PS、DCF）都从这个公式推导出来的。

```
价值 = CF₁/(1+r)¹ + CF₂/(1+r)² + ... + CFₙ/(1+r)ⁿ + 终值/(1+r)ⁿ
```

三个变量决定价值：
1. **CF（现金流）** — 这家公司未来能赚多少钱？
2. **r（折现率）** — 你对这笔投资要求的回报率是多少？
3. **n（持续时间）** — 超额收益能持续多少年？

**估值不是算一个精确数字，而是判断这三个变量的方向和量级。**

## 二、DCF 作为思维框架（不是计算器）

很多人不学 DCF，觉得"太复杂、假设太多、算出来也不准"。

**这是对的——如果你把 DCF 当计算器用。**

但如果你把 DCF 当**思维框架**用，它是最强大的估值工具。因为 DCF 强迫你思考三件事，而这三件事恰恰是 PE 忽略了的事：

### DCF 强迫你思考的三个问题

**问题 1：利润能变成现金吗？**

PE 只看利润（Earnings），但当一家公司利润增长 20% 但经营现金流不涨时（应收款堆起来了），PE 看不出来。DCF 从自由现金流出发，你首先要问的是——**"净利润里有多少是真正能拿到手的？"**

→ 这就是为什么 [[financial-analysis-foundation]] 要放在估值前面。你不先搞清利润质量，估值就是 garbage in garbage out。

**问题 2：增长需要多少资本投入？**

一家公司想增长 10%，需要再投多少钱？如果维持性资本开支吃掉了增长的大部分现金流，那这种增长对股东没太大意义。PE 看的是利润，DCF 看的是"利润减去维持增长必须花的钱"之后剩下的。

→ 你可以理解为什么电力股 PE 低：因为增长需要巨额资本投入（建新电厂），自由现金流远小于利润。

**问题 3：终值占比 70%+，你押的是什么？**

任何一个 DCF 模型里，前三年的现金流只占总价值的 15-30%，剩下的 70-85% 来自终值——也就是"三年以后的永续价值"。这意味着：

**估值本质上不是预测未来三年，而是判断这家公司"长期会怎样"。**

而终值取决于两个变量：永续增长率（g）和折现率（r）。这个 g 从 2% 改成 3%，价值可能差 30%。所以最关键的问题不是"明年增速多少"，而是"10 年后这家公司还在不在？还能不能增长？"

→ 芒格的框架（护城河持久性）在这里直接和估值挂钩了。

### DCF 思维框架的实操用法

不用建 Excel。拿到任何一家公司，脑子里跑这三个问题：

```
1. 自由现金流 ≈ 经营现金流 − 维持性资本开支 ≈ 净利润的百分之几？
2. 未来 5-10 年增速大概在什么区间？（用行业和护城河趋势判断）
3. 10 年后这个公司还在吗？还能增长吗？（护城河宽度决定答案）
```

三个答案决定了你愿意为它付多少钱。

## 三、为什么不同行业用不同倍数的底层逻辑

手册里有口诀（银行看 PB、消费看 PE、科技看 PS），但没有解释为什么。知其所以然，才知道什么情况下口诀会失效。

### 3.1 银行：看 PB，不看 PE

**为什么 PE 不可靠？**

银行的利润是"做出来"的——管理层可以通过调节拨备（多提或少提）在很大范围内操纵净利润。拨备覆盖率从 292% 降到 200%，利润就多了几百亿，而实际经营没有任何变化。

所以银行的 PE 高低，可能只是因为拨备政策的差异，不代表任何价值信息。

**为什么 PB 更可靠？**

银行的资产主要是贷款和债券，在资产负债表上以公允价值或摊余成本计量，**账面净资产大致反映了清算价值**。

PB = 0.8 意味着什么？市场认为农行的真实净资产打 8 折——因为有些贷款可能收不回来，但账面还没体现。

**农行当前案例**：
```
农行 PB = 0.81
→ 市场给净资产打 19% 的折扣
→ 这个折扣隐含了什么？
→ 如果真实不良率是 5%（不是账面 1.27%），净资产确实要打折
→ 所以判断农行是贵还是便宜，核心不是看 PB 数字，是判断"真实不良率"比账面高多少
→ 如果拨备覆盖 293%，意味着即使不良率翻倍，拨备也够覆盖 → PB 0.81 可能低估了
```

**银行的快速估值判断**：
```
合理 PB ≈ 1 − (真实不良率 − 账面不良率) × (1 − 回收率)

如果真实不良率 = 账面不良率（即不良认定严格）→ PB 应该接近 1.0
如果真实不良率远高于账面 → PB 应该远低于 1.0

农行不良认定：逾期 90 天以上/不良 = 71.95%（< 100% 即非常严格）
→ 不良认定很严 → PB 0.81 可能偏低估
```

### 3.2 消费：看 PE + PEG

**为什么 PE 对消费股有效？**

消费品公司的利润有几个特征：
- 现金流好（预收款模式，利润 ≈ 现金）
- 利润可预测（品牌壁垒下的稳定增长）
- 不需要大量再投资（轻资产，维持性资本开支低）

这三个特征让 PE 成为一个合理的估值工具——利润数字可靠，增速可预测，不需要像银行那样调整。

**但 PE 不是孤立的**，必须跟增速配对：
```
PEG = PE / 净利润增速

PEG < 1：可能低估
PEG = 1-1.5：合理
PEG > 2：可能高估
```

**海天当前案例**：
```
当前 PE(TTM)：26.78 倍（2026.6.17）
2025 年净利润增速：10.95%
PEG = 26.78 / 10.95 ≈ 2.45 → 偏贵？

等一下——
问题：海天的"真实增速"是多少？
2025 年：+10.95%（含 H 股上市费用，扣非 +12.81%）
2026 年一致预期：+11.8%

PEG ≈ 27 / 12 ≈ 2.25 → 仍然偏贵

但为什么市场愿意给海天 25-30 倍 PE？（而一般食品公司可能只值 15-20 倍）
→ 因为海天有护城河溢价：品牌 + 渠道垄断 + 22-25% 净利率（同业无人能及）
→ 所以 25-30 倍可能是海天的"合理估值中枢"，不能用一般消费品的标准
→ 这是护城河框架直接用于估值判断
```

**消费股估值的关键洞察**：
- 同一行业里，**PE 差距反映的是护城河宽度的差距**
- 海天 PE 26 倍 vs 一般调味品公司 PE 15 倍 → 差出来的 11 倍是护城河溢价
- 所以判断"贵不贵"时，要和它自己的历史 PE 中枢比，不是和行业平均比

**海天历史 PE 参考**：
```
10 年 PE 中枢：25-30 倍（当前 26.78 倍，处于中枢）
5 年 PE 中枢：35-40 倍（当前低于中枢，因为 2021 年泡沫吹高了 5 年线）
峰值 PE：> 100 倍（2021 年泡沫期）
谷底 PE：~23-25 倍

当前判断：处于合理水平，不是便宜到"闭眼买"，也不是贵到"不敢看"
         → 不值得重仓追，但也不用恐慌卖出
```

### 3.3 科技/成长：PE 基本没用

科技股（如 CPO）的利润高度不稳定——可能在爆发初期亏损，利润爆发后又面临技术迭代风险。114 倍 PE 不能告诉你任何东西，因为"E"本身在剧烈变化。

**科技股可行的估值方法**：
- **PS（市销率）**：当利润为负或不稳定时，看收入质量
- **PEG（看前瞻增速而非历史）**：如果明年增速 100%，PE 100 倍 = PEG 1.0
- **终局思维**：算一下 5 年后这个市场多大、这家公司能拿多少份额、能赚多少利润 → 倒推今天值多少

## 四、反向 DCF——第二层思维的量化工具

这是最实用的估值工具，但很少人用。

### 是什么？

常规 DCF："假设增速 13%，折现率 8% → 价值 = 100 元"

**反向 DCF**："当前价格 100 元，折现率 8% → 市场在假设什么增速？"

把问题从"我觉得这家值多少钱"变成"**当前价格在赌什么样的未来**"——然后你判断这个"赌注"合不合理。

### 实操：海天当前价格隐含了多少增长？

```
当前股价：33.32 元
当前 2025 EPS：70.38 亿 / 58.52 亿股 ≈ 1.20 元/股
当前 PE：33.32 / 1.20 ≈ 27.8 倍

简化反向 DCF（假设折现率 8%）：
→ 27.8 倍 PE 隐含未来 5 年复合增速约多少？

大致规律：
PE 10 倍 → 隐含 0-2% 永续增长
PE 15 倍 → 隐含 3-5% 增长
PE 20 倍 → 隐含 5-7% 增长
PE 27 倍 → 隐含 8-10% 增长
PE 50 倍 → 隐含 15%+ 增长
PE 100 倍 → 隐含 20%+ 增长

海天 PE 27 倍 → 市场隐含海天未来 5-10 年增速 8-10%
→ 海天 2025 实际增速 11%，2026 预期 12%
→ 市场给的假设略低于当前实际增速
→ 这意味着：如果海天能维持 10-12% 增速，当前价格偏合理偏低；如果掉到 5-7%，就贵了

再对照历史：海天 2015-2025 十年净利润 CAGR 约 14%
→ 当前价格隐含的增速低于历史均值
→ 从反向 DCF 角度看，当前估值不算贵
```

### 实操：CPO 当前 PE 隐含了什么？

（用之前 CPO 讨论的数据做思维练习）

```
中际旭创 PE = 114 倍（TTM）
→ 简化反向 DCF：隐含未来 5-10 年增速 20%+

问题：光模块行业未来 5-10 年能维持 20%+ 增长吗？
→ 可能可以（AI 算力需求 + 光模块升级周期）
→ 但 114 倍 PE 有安全边际吗？
→ 没有——任何低于 20% 增长的情景都会导致估值收缩
→ 所以风险收益比不对称：对了赚估值回归的钱，错了亏估值+业绩双杀

→ 这也解释了为什么 6 月初 CPO 急跌：估值太满，风吹草动就崩
```

### 反向 DCF 的通用规律

| PE（假设折现率 8%） | 隐含的永续增长率 |
|---------------------|-----------------|
| 5-8 倍 | 0-1% |
| 10-12 倍 | 2-3% |
| 15-18 倍 | 5-6% |
| 20-25 倍 | 7-9% |
| 30-40 倍 | 10-13% |
| 50-70 倍 | 15-18% |
| 100 倍+ | 20%+ |

⚠️ 这是极其简化的估算，不同行业、不同折现率会不一样。但作为"第二层思维"的起点足够用了。

**用法**：看到一只股票的 PE → 查这张表 → 问自己"这个增速合理吗？" → 答案就是你的估值判断。

## 五、三种估值方法的适用矩阵

| 方法 | 适用行业 | 不适用行业 | 核心问题 |
|------|---------|----------|---------|
| **PE** | 消费、医药、稳定增长的制造 | 银行（利润可操纵）、科技（利润不稳定）、周期（利润波动大） | "利润质量好吗？增速可持续吗？" |
| **PB** | 银行、保险、公用事业、重资产周期股 | 轻资产服务、科技、品牌消费品 | "净资产真实吗？有隐藏不良吗？" |
| **反向 DCF** | 任何行业 | 无（它只是一个思考框架） | "当前价格在赌什么样的未来？合理吗？" |

## 六、估值的十条铁律

1. **估值是区间，不是点。** 没有人能算出一个精确的"内在价值"。给一个合理区间就够了。

2. **PE 告诉你"相对贵不贵"；DCF 告诉你"值不值"。** 两者结合用。

3. **不要拿不同行业的 PE 比。** 银行 PE 8 倍 vs 海天 PE 27 倍 —— 不是银行便宜，是不同行业的利润质量不同。

4. **一家公司的合理 PE，要跟自己比（历史中枢），不要跟别人比。**

5. **终值占 DCF 70%+。** 所以最重要的判断是"这家公司 10 年后还在不在"——不是明年增速多少。

6. **反向 DCF 比正向 DCF 实用 10 倍。** 你不用猜未来，你只需要判断"价格在赌什么"。

7. **高增长掩盖一切估值问题，低增长暴露一切估值问题。** 所以不要在高速增长期追高——你不知道增速会不会突然掉下来。

8. **便宜有便宜的道理，贵有贵的道理。** 不要因为"便宜"就买——先搞清楚为什么便宜。A 股破净的银行股很多，但不是每个都值得买。

9. **股息率是最诚实的估值信号。** 利润可以造假，分红不行。高股息率 + 可持续分红 = 真便宜。

10. **最好的估值工具是耐心。** 等待市场犯错——恐慌时 PE 被压到历史低位，狂热时 PE 被推到历史高位。在低位买，你的安全边际就来自别人的情绪。

本手册是 [[investment-handbook]] 维度三（估值水位）的深度展开。结合使用时：

1. 先用 [[financial-analysis-foundation]] 确认利润质量
