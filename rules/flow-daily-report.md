# 日报生成流程

> 触发词：今天怎么样了/日报/更新日报

## 第〇步：确认时间和交易日

- 今天是不是交易日？（不是 → 告知休市，不生成）
- 现在几点？
  - 9:30 前 → 盘前，只有昨日数据
  - 9:30-15:00 → 盘中，用实时数据
  - 15:00-17:00 → 刚收盘，数据不全
  - 17:00 后 → 最佳时间

## 第一步：并行拉数据

用 akshare MCP 同时获取（不顺序拉）：
- 持仓股票：601288(农行) / 603288(海天) / 561560(电力ETF) / 600377(宁沪) — 股价+涨跌幅
- 上证指数 + 成交额 + 领涨领跌板块
- 重大政策/事件（WebSearch）

## 第二步：加载模板

Read `knowledge/06-执行层/investment-daily-check.md` → 取持仓结构 + 止损线

## 第三步：框架分析（不是列数字）

- **市场温度**：加载 `02-宏观层/financial-crisis-classics.md`，判断当前处于 Kindleberger 哪个阶段
- **宏观传导**：加载 `02-宏观层/macro-transmission.md`，今天的宏观事件怎么影响持仓？
- **止损预警**：距止损 < 10% 的持仓，必须分析"为什么跌 + 怎么办"
- **每只归因**：引用对应行业框架解释表现

## 第四步：事前验尸

当前组合最大的脆弱点在哪？用一个具体场景描述。

## 第五步：写日志

追加到 `knowledge/08-日志/investment-daily-log.md`（文件顶部插入新条目）。

## 第六步：生成摘要

写 ~50 行桌面摘要，只包含：一句话总结 + 每只一句话 + 今日操作建议。

## 耗时目标

~8 分钟（框架加载 + WebSearch 并行 + 分析 + 生成）
